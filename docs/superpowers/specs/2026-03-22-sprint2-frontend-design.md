# Sprint 2 Frontend — The Mine Home Screen Design

**Goal:** 홈 화면(The Mine)에서 광맥 3개를 보고, 선택하고, 아이디어 10개를 생성하고, 가방에 담아 Vault로 반입하는 전체 프론트엔드 흐름.

**Architecture:** Expo Router 탭 구조의 `index.tsx`(The Mine)를 중심으로, 백엔드 3개 API와 연결. 상태 관리는 React 로컬 상태로 충분 (Zustand 불필요).

## 2. 컴포넌트 구조

### 2-1. 상태바 (StatusBar)

2줄 고정 상태바.

**프로필 줄:** 닉네임 + 광부 레벨 + 연속 접속
**자원 줄:** 채굴 잔여 + 리롤 잔여 + 가방 용량 + 보유 재화

```
⛏ 도민 광부님  Lv.3    🔥 연속 3일
채굴 1/1  🔄 2/2  🎒 0/2    💎 120
```

### 2-2. 미니 광맥 카드 (MiniVeinCard)

3개가 가로로 나란히 배치. 탭하면 선택 상태 + 아래에 확장.

**표시 정보:**
- 희귀도 (일반 / 반짝✦ / 희귀★)
- 키워드 1~2개 미리보기 (잘림)

**선택 상태:** 테두리 골드 하이라이트

### 2-3. 확장 광맥 카드 (ExpandedVeinCard)

선택된 광맥의 상세 정보.

**표시 정보:**
- 광맥 이름 (자동 생성)
- 희귀도
- 키워드 칩 전체 (도트 인디케이터 + 카테고리별 컬러)
- "채굴하기" CTA 버튼

### 2-4. 키워드 칩 (KeywordChip)

카테고리별 컬러 도트 + 키워드 이름.

**카테고리별 컬러:**

| 카테고리 | 도트 컬러 |
|----------|----------|
| AI | `#8B5CF6` (보라) |
| Who | `#EC4899` (핑크) |
| Domain | `#4E9A6B` (초록) |
| Tech | `#6496FF` (파랑) |
| Value | `#C9A044` (골드) |
| Money | `#B85450` (빨강) |

### 2-5. 원석 카드 (IdeaCard)

10개 리스트의 각 아이템.

**표시 정보:**
- 제목
- 요약 (2~3줄)
- 사용된 키워드 (도트 칩)
- "가방에 담기" / "광차에 싣기" 버튼 (티어에 따라)

**담긴 상태:** 체크 표시 + 버튼이 "빼기"로 변경

### 2-6. 리롤 버튼 (RerollButton)

"⛏ 다시 파기 (N/N)" — 잔여 횟수 표시.
소진 시 비활성화 + "광고로 추가 리롤" 안내.

### 2-7. 하단 반입 버튼 (VaultButton)

원석 결과 화면 하단에 고정. "Vault로 반입하기 (N)" — 담긴 수 표시.
0개면 비활성화.

## 3. 운반 시스템

| 티어 | 운반 도구 | 용량 | 버튼 텍스트 |
|------|----------|------|-----------|
| Free | 가방 (Bag) | 2개 (레벨업 → 최대 5개) | "가방에 담기" |
| Lite/Pro | 광차 (Mine Cart) | 10개 전부 | "광차에 싣기" |

- Free: 가방이 가득 차면 나머지 "가방에 담기" 버튼 비활성화
- Lite/Pro: 전부 담을 수 있으므로 제한 없음
- 이미 담은 건 다시 눌러서 빼기 가능

## 4. 데이터 흐름

```
홈 화면 로드
  ↓ GET /mining/veins/today
  ↓ (광맥 3개 + 일일 상태)
미니 카드 3개 표시
  ↓ 유저가 광맥 1개 탭
확장 카드 표시
  ↓ "채굴하기" 탭
  ↓ POST /mining/veins/{id}/mine
원석 10개 결과 화면
  ↓ 유저가 "가방에 담기" (로컬 상태)
  ↓ "Vault로 반입하기" 탭
  ↓ PATCH /ideas (is_vaulted = true) — 선택된 아이디어만
홈 화면으로 복귀
```

### 리롤 흐름

```
"다시 파기" 탭
  ↓ POST /mining/veins/reroll
미니 카드 3개 갱신 + 리롤 카운터 감소
```

## 5. API 연결

| 엔드포인트 | 시점 | 응답 |
|-----------|------|------|
| `GET /mining/veins/today` | 홈 화면 로드 | 광맥 3개 + 키워드 + 일일 상태 |
| `POST /mining/veins/reroll` | 리롤 버튼 | 새 광맥 3개 + 리롤 카운터 |
| `POST /mining/veins/{id}/mine` | 채굴하기 | 아이디어 10개 |

추가 필요한 API (현재 미구현):
- `PATCH /ideas/{id}` — is_vaulted 변경 (가방에 담기 → Vault 반입)
