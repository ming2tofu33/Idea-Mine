# IDEA MINE 프로젝트 문서 v0.1

## 1. 프로젝트 한 줄 정의
**IDEA MINE**은 AI 기술, 타겟, 산업, 가치, 수익모델을 조합해 기획 아이디어를 발굴하고 발전시키는 **픽셀 아트 기반 아이디어 제너레이션 플랫폼**이다.

---

## 2. 프로젝트 개요

### 2-1. 문제 정의
AI 기술은 빠르게 늘어나고 있지만, 기획자는 다음 문제를 반복해서 겪는다.
- 어떤 기술을 어떤 타겟에게 붙여야 할지 감이 잘 오지 않는다.
- 아이디어가 있어도 비즈니스 가치와 수익 모델까지 연결하기 어렵다.
- 떠오른 아이디어가 일회성 메모로 끝나고 자산화되지 않는다.
- 좋은 조합을 찾기까지 시간이 오래 걸리고, 실패한 시도는 기록되지 않는다.

### 2-2. 해결 방식
IDEA MINE은 아이디어 발상을 **광산 채굴 메타포**로 단순화한다.
- 키워드 탐색 = 채굴
- 조합 실험 = 제련
- 개요서 발전 = 가공
- 공개 전시 = 쇼케이스
- 투자자 연결 = 거래소/매칭

즉, 사용자는 막연한 브레인스토밍이 아니라 **조합 가능한 원석 체계**를 바탕으로 빠르게 아이디어를 만들고, 발전시키고, 저장하고, 보여줄 수 있다.

### 2-3. 핵심 가치 제안
- **빠른 발상:** 빈 화면에서 시작하지 않게 해준다.
- **구조화된 기획:** Tech, Who, Domain, Value, Money를 한 번에 보게 한다.
- **아이디어 자산화:** 생성 → 개요서화 → 보관 → 전시까지 이어진다.
- **차별적 경험:** 픽셀 아트와 광산 세계관으로 감성적 몰입감을 만든다.

## 5. 정보 구조 (세계관 기반)

### 5-1. 공간 구조
#### 1) The Mine
키워드를 발견하는 탐색 공간
- 오늘의 추천 조합
- 랜덤 채굴
- 리롤
- 트렌드 키워드 노출

#### 2) The Lab
키워드를 조합하고 실험하는 공간
- 직접 키워드 선택
- 슬롯머신 조합
- AI 아이디어 생성
- 커스텀 키워드 관리

#### 3) The Vault
아이디어와 개요서를 보관하는 공간
- 저장한 아이디어 목록
- 상세 개요서 보관
- 북마크 / 폴더 / 태그

#### 4) The Showcase
아이디어를 공개 전시하는 공간
- 티저 전시
- 인기 아이디어
- 투자자 노출용 카드

#### 5) The Exchange
아이디어와 투자자를 연결하는 공간
- 투자자 필터
- 관심 키워드 설정
- 매칭/연락 기능

#### 6) The Observatory
상위 지표 및 인사이트를 보는 공간
- 어떤 조합이 많이 생성되는지
- 어떤 가치/기술이 인기인지
- 투자 관심 분포

## 10. 핵심 의사결정 요약
- 본체 루프는 **광맥 → 아이디어 원석 → 금고 → 실험실**이다.
- 실험실은 **프로젝트 개요 → 감정 → 방향 정리 → MVP 청사진 → Phase 로드맵**으로 확장된다.
- 금고는 **저장 / 회수 / 실험실 연결**에 집중한 MVP 구조로 시작한다.
- Free / Lite / Pro는 **사용 횟수보다 문서 깊이와 작업 해상도**에서 차이를 만든다.
- 악성 입력과 공개 악용은 **Clean Mine Protocol**으로 관리하고, 성실한 활동은 **배지 기반 신뢰도 시스템**으로 보상한다.
- 메타 게임 요소는 장기적으로 가능하지만, **세부 경제와 꾸미기 설계는 별도 문서**에서 관리한다.

## 13. 메타 게임 확장 방향 (상위 개념만)

IDEA MINE은 장기적으로 **가벼운 게임 메타 루프**를 붙여 재방문성과 애착을 강화할 수 있다.

확장 가능한 메타 요소:
- 광산 / 방 / 실험실 꾸미기
- 광부 캐릭터 꾸미기
- 가벼운 케어 요소
- 시즌성 / 희귀 보상형 아이템

원칙:
- 메타 루프는 아이디어 기획 흐름을 방해하지 않는다.
- 꾸미기와 성장 요소는 숙제가 아니라 보너스 감정 루프로 작동해야 한다.
- 패널티형 관리보다 애착과 수집 욕구를 자극하는 방향이 적절하다.
- 재화 구조, 꾸미기 상점, 희귀 보상, 밸런스 수치 등은 **별도 게임 경제/메타 루프 설계 문서**로 분리 관리한다.
