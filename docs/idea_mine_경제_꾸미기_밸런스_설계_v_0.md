# IDEA MINE 경제·꾸미기 밸런스 설계 v0.1

## 1. 문서 목적
이 문서는 IDEA MINE의 메인 제품 문서와 분리된 **게임 경제 / 꾸미기 / 메타 루프 전용 설계 문서**다.

메인 문서는 핵심 제품 구조, IA, 핵심 루프, 유료화, 실험실 문서 체계를 다루고,
이 문서는 아래 영역을 별도로 다룬다.
- 꾸미기 재화 구조
- 메타 보상 루프
- 공간/캐릭터 성장 방향
- 희귀 보상 구조
- 경제 밸런스 원칙
- Phase별 도입 순서

이 영역은 밸런싱과 운영 난이도가 높기 때문에,
메인 PRD에 모두 합치기보다 별도 문서로 관리하는 것이 적절하다.

---

## 2. 기본 방향
IDEA MINE의 메타 게임 요소는 본체를 대체하는 것이 아니라,
**아이디어 채굴 활동에 보상을 붙여 재방문성과 애착을 강화하는 보조 루프**로 설계한다.

즉,
- 본체 루프 = 광맥 → 아이디어 원석 → 금고 → 실험실
- 메타 루프 = 채굴/문서화 보상 → 재화 획득 → 공간/캐릭터 성장

### 2-1. 핵심 원칙
1. 꾸미기와 케어는 메인 기획 흐름을 방해하지 않는다.
2. 메타 보상은 채굴과 문서화 활동에 자연스럽게 연결된다.
3. 관리 요소는 숙제가 아니라 애착과 보상 루프로 작동해야 한다.
4. 재화 종류는 최소화하고 역할은 분명하게 나눈다.
5. 희귀 보상은 드물게 등장해 발견감을 만들어야 한다.

---

## 3. 메타 루프의 역할
IDEA MINE의 경제/꾸미기 시스템은 아래 4가지 감정을 보조해야 한다.

### 3-1. 축적의 재미
채굴하고 문서화할수록 내 광산과 작업 공간이 자란다는 감각.

### 3-2. 애착의 재미
내 캐릭터, 내 작업실, 내 실험실이라는 감각.

### 3-3. 발견의 재미
가끔 특별한 보상이 나오고, 희귀한 장식과 재화를 발견하는 재미.

### 3-4. 돌봄의 재미
가벼운 케어 요소를 통해 캐릭터와 공간에 애착을 붙이는 재미.
단, 패널티형 의무 관리로 가지 않는다.

## 4. 재화 구조
현재 기준에서 가장 적절한 구조는 **2통화 + 성장 자산 보조 구조**다.

### 4-1. 기본 꾸미기 재화: 핑크 결정
**핑크 결정**은 IDEA MINE의 기본 꾸미기 재화다.

#### 역할
- 채굴/문서화 활동으로 꾸준히 모이는 기본 재화
- 광산/작업실/실험실 꾸미기 구매 재화
- 광부 캐릭터 기본 꾸미기 구매 재화
- 가벼운 케어 아이템 구매 재화

#### 사용자 감정
- 열심히 캐면 내 공간이 자란다
- 조금씩 모아서 내 광산을 꾸민다

즉, 핑크 결정은 **노력형 / 축적형 / 일상형 재화**다.

### 4-2. 프리미엄 재화: Pink Diamond
**Pink Diamond**는 프리미엄 재화다.

#### 역할
- 인앱 결제 중심 재화
- 희귀 장식, 한정 스킨, 특수 확장권 구매
- 광고 대체 및 프리미엄 선택권 제공
- 작업권 구매 및 고급 꾸미기 해금

#### 사용자 감정
- 내 광산을 더 특별하게 만든다
- 희귀하고 아껴 쓰는 재화다

즉, Pink Diamond는 **프리미엄형 / 희귀형 / 선택형 재화**로 본다.

### 4-3. 랜덤 희귀 재화: 고대 동전
**고대 동전**은 낮은 확률로 획득하는 희귀 보상형 재화다.

#### 역할
- 랜덤 드롭 또는 이벤트 보상
- 고대 교환소 전용 재화
- 희귀 장식, 시즌 아이템, 특별 보상 교환

#### 사용자 감정
- 오, 이건 드문 보상이다
- 특별한 걸 바꿀 수 있는 보너스다

즉, 고대 동전은 **랜덤 보상형 / 이벤트형 / 희귀 교환 재화**로 정의한다.

### 4-4. 성장 자산: 반입 슬롯
반입 슬롯은 통화가 아니라 **성장 자산**이다.

#### 역할
- 한 번의 채굴에서 금고로 가져갈 수 있는 원석 수를 결정
- 광부 레벨과 성장 보상 구조를 연결

#### 초기 기준
- 레벨 1: 원석 10개 중 2개 반입 가능
- 이후 레벨업에 따라 3개 → 4개 → 5개 확장

#### 사용자 감정
- 더 많이 저장하는 것이 아니라 더 많이 회수할 권한이 생긴다

## 5. 핑크 결정 획득처
핑크 결정은 **아이디어 활동이 곧 꾸미기/메타 보상으로 이어진다**는 감각을 주어야 한다.

### 5-1. 핵심 획득처
- 광맥 채굴 완료
- 아이디어 원석 금고 반입
- 프로젝트 개요 생성
- 감정 완료
- 데일리 퀘스트 완료
- 연속 채굴 보상
- 광부 레벨업 보상
- 배지 획득 보상

### 5-2. 추가 획득처
- 주말 특별 광맥 보너스
- 취향 광맥 첫 발견 보상
- 첫 채굴 / 첫 개요 / 첫 감정 같은 마일스톤 보상

### 5-3. 핑크 결정 획득 문구
#### 기본 획득
- `핑크 결정을 획득했어요`
- `반짝이는 핑크 결정이 추가됐어요`
- `채굴 보상으로 핑크 결정을 얻었어요`

#### 채굴 완료 후
- `채굴 보상: 핑크 결정 +20`
- `오늘의 광맥에서 핑크 결정을 회수했어요`
- `원석 채굴이 끝나고 핑크 결정이 남았어요`

#### 금고 반입 시
- `원석 반입 보상으로 핑크 결정을 얻었어요`
- `좋은 원석을 회수해 핑크 결정이 쌓였어요`

#### 프로젝트 개요 생성 시
- `프로젝트 개요를 완성해 핑크 결정을 획득했어요`
- `아이디어를 문서로 다듬어 핑크 결정이 지급됐어요`

#### 감정/실험실 활동 시
- `실험실 활동 보상으로 핑크 결정이 추가됐어요`
- `감정을 마치고 핑크 결정을 얻었어요`

#### 데일리 퀘스트 보상 시
- `데일리 퀘스트 완료! 핑크 결정을 받았어요`
- `오늘의 작업을 마치고 핑크 결정이 지급됐어요`

## 6. 핑크 결정 사용처
핑크 결정은 너무 많은 시스템에 쓰지 말고,
**눈에 보이는 만족감이 있는 소비처**에 집중하는 것이 좋다.

### 6-1. 작업실 / 광산 꾸미기
- 바닥
- 벽지
- 조명
- 책상
- 의자
- 모니터
- 광산 입구 장식
- 작업용 장식품

### 6-2. 실험실 꾸미기
- 실험 테이블
- 스캐너 외형
- 분석 장비 스킨
- 데이터 패널 배경
- 제련 장치 장식

### 6-3. 금고 꾸미기
- 금고 외형
- 진열대
- 보관함 스킨
- 보안 패널 외형
- 원석 전시 슬롯 꾸미기

### 6-4. 광부 기본 꾸미기
- 안전모
- 작업복
- 안경
- 곡괭이 기본 스킨
- 장갑 / 배낭
- 배지 프레임

### 6-5. 가벼운 케어 아이템
- 간식
- 음료
- 휴식 아이템
- 반응 유도 소품

#### 운영 원칙
- 케어는 패널티 없는 보너스 감정 루프로 유지한다.
- 안 챙겼다고 손해 보게 하지 않는다.

## 7. Pink Diamond 사용처
Pink Diamond는 기능 확장과 프리미엄 꾸미기에 집중한다.

### 7-1. 프리미엄 꾸미기
- 희귀 가구
- 한정 캐릭터 스킨
- 시즌 장식
- 특별 광맥 테마 장식
- 금고 외형 업그레이드
- 실험실 프리미엄 장식

### 7-2. 편의 / 확장
- 꾸미기 슬롯 확장
- 금고 확장
- 취향 광맥 부스터
- 희귀 광맥 티켓
- 고급 테마 해금

### 7-3. 기능 확장권 구매
- 야간 작업권
- 개요 확장권
- 정밀 감정권
- 청사진 확장권

즉, Pink Diamond는 **프리미엄 꾸미기 + 기능 확장** 중심 재화다.

## 8. 고대 동전 구조
고대 동전은 기본 재화가 아니라,
희귀하게 등장하는 **랜덤 보너스형 수집 재화**로 운영한다.

### 8-1. 획득 방식
- 채굴 중 낮은 확률로 랜덤 드롭
- 주말 특별 광맥 보상
- 취향 광맥 보너스
- 특정 이벤트 / 시즌 보상
- 레벨 마일스톤 / 배지 마일스톤의 희귀 보상

### 8-2. 획득 문구
#### 기본 획득
- `고대 동전을 발견했어요`
- `희귀 보상! 고대 동전이 나왔어요`
- `광맥 깊은 곳에서 고대 동전을 회수했어요`

#### 랜덤 드롭
- `반짝! 채굴 중 고대 동전이 발견됐어요`
- `운 좋게 고대 동전이 나왔어요`
- `이번 채굴에서 희귀 보상인 고대 동전을 얻었어요`

#### 특별 광맥 보상
- `특별 광맥 보상으로 고대 동전을 획득했어요`
- `주말 광맥 깊은 곳에서 고대 동전을 찾았어요`

#### 취향 광맥 보상
- `취향 광맥 보너스! 고대 동전이 추가됐어요`
- `당신의 광맥에서 희귀한 고대 동전이 나왔어요`

### 8-3. 사용처
고대 동전은 일반 상점이 아니라 **고대 교환소**에서만 사용한다.

#### 추천 교환 품목
- 희귀 장식
- 한정 스킨
- 시즌 아이템
- 특별 광맥 테마 장식
- 프레임 / 명패 / 희귀 배경

즉, 고대 동전은 **수집 감각과 희귀 보상감**을 만드는 재화다.

## 9. 상점 구조
초기 상점 구조는 복잡하게 나누지 않고 2축으로 가는 것이 좋다.

### 9-1. 꾸미기 상점
핑크 결정 중심 상점

#### 상단 설명 문구
- `채굴로 모은 핑크 결정으로 광산을 꾸며보세요`
- `핑크 결정은 꾸미기와 기본 아이템 구매에 사용할 수 있어요`
- `반짝이는 핑크 결정으로 당신만의 공간을 채워보세요`

#### 구매 버튼 문구
- `핑크 결정으로 구매`
- `결정으로 교환`
- `이 장식 가져오기`

#### 부족 문구
- `핑크 결정이 조금 더 필요해요`
- `결정이 부족해 아직 가져올 수 없어요`
- `조금만 더 채굴하면 이 장식을 가져올 수 있어요`

#### 구매 완료 문구
- `새 장식이 광산에 추가됐어요`
- `핑크 결정으로 아이템을 교환했어요`
- `당신의 공간에 새 꾸미기 아이템이 놓였어요`

### 9-2. 고대 교환소
고대 동전 전용 희귀 상점

#### 상단 설명 문구
- `희귀한 고대 동전으로 특별한 아이템을 교환할 수 있어요`
- `고대 동전은 깊은 광맥에서만 가끔 발견돼요`
- `고대 교환소에서는 한정 장식과 특별한 보상을 만날 수 있어요`

#### 교환 버튼 문구
- `고대 동전으로 교환`
- `희귀 교환하기`
- `이 유물 가져오기`

#### 부족 문구
- `고대 동전이 부족해요`
- `이 아이템은 더 많은 고대 동전이 필요해요`
- `희귀 교환에는 고대 동전이 더 필요해요`

#### 교환 완료 문구
- `고대 교환이 완료됐어요`
- `희귀 아이템을 손에 넣었어요`
- `고대 동전으로 특별한 보상을 교환했어요`

## 10. 꾸미기 카테고리 구조
꾸미기는 일반 하우징 게임처럼 넓게 가지 않고,
IDEA MINE의 본체 루프와 연결되는 공간만 꾸미게 하는 구조가 적절하다.

### 10-1. 작업실
가장 중심이 되는 개인 베이스캠프.

역할:
- 가장 자주 돌아오는 공간
- 사용자의 취향과 기본 무드를 드러내는 공간

추천 요소:
- 바닥, 벽, 조명
- 책상, 의자, 모니터
- 선반, 소형 장식, 네온 사인

### 10-2. 실험실
IDEA MINE의 차별점이 가장 잘 살아나는 공간.

역할:
- 아이디어를 정제하는 상징 공간
- 기획과 제련의 감각을 강화하는 공간

추천 요소:
- 분석 장비
- 실험 테이블
- 데이터 패널
- 스캐너, 제련 장치, 홀로그램 장식

### 10-3. 금고
보관과 수집의 감각을 주는 공간.

역할:
- 내가 쌓아둔 성과가 보이는 공간
- 꾸미기이면서 동시에 트로피룸 역할

추천 요소:
- 금고 외형
- 선반, 진열대, 보관함
- 희귀 원석 전시 슬롯
- 프로젝트 명패, 배지 진열판

### 10-4. 광부 / 장비
사용자 애착의 핵심인 아바타 영역.

역할:
- 자아투영
- 수집과 스타일 표현

추천 요소:
- 안전모, 작업복, 안경
- 곡괭이, 장갑, 배낭
- 배지 프레임, 칭호 장식

## 11. 꾸미기 레이어 구조
각 꾸미기 카테고리는 아이템 종류보다 먼저 **레이어 구조**를 정의하는 것이 중요하다.

### 11-1. 공통 레이어
#### 1) 배경 레이어
벽, 바닥, 컬러 테마, 배경 패널

#### 2) 구조물 레이어
책상, 선반, 실험 테이블, 금고 프레임 같은 큰 구조물

#### 3) 핵심 오브젝트 레이어
모니터, 분석 장비, 중앙 금고, 메인 곡괭이 같은 대표 장치

#### 4) 보조 소품 레이어
머그컵, 노트, 공구함, 광석 병, 파일 박스 같은 작은 디테일 요소

#### 5) 장식 / 성과 레이어
포스터, 트로피, 희귀 원석 전시, 배지 액자, 한정 장식

#### 6) 이펙트 / 분위기 레이어
조명 효과, 홀로그램, 반짝임, 증기, 픽셀 먼지 등 분위기 강화 요소

### 11-2. 작업실 레이어 예시
- 배경: 벽지, 바닥, 창문, 조명 테마
- 구조물: 책상, 의자, 선반, 러그
- 핵심 오브젝트: 모니터, 키보드, 메인 조명
- 보조 소품: 메모지, 커피잔, 작은 화분, 케이블
- 장식: 포스터, 네온 사인, 배지 액자
- 이펙트: 모니터 빛, 창밖 무드, 네온 점멸

### 11-3. 실험실 레이어 예시
- 배경: 실험실 벽면, 타일 바닥, 데이터 패널
- 구조물: 실험 테이블, 장비 선반, 스캔 프레임
- 핵심 오브젝트: 제련 장치, 원석 스캐너, AI 분석 콘솔
- 보조 소품: 샘플 병, 데이터 칩, 실험 노트
- 장식: 희귀 원석 샘플, 실험 기록 패널
- 이펙트: 홀로그램 빛, 스캔선, 경고등

### 11-4. 금고 레이어 예시
- 배경: 금속/돌 벽면, 보안 패널
- 구조물: 보관 선반, 진열대, 보관함
- 핵심 오브젝트: 메인 금고 문, 중앙 원석 전시대
- 보조 소품: 라벨 카드, 파일 폴더, 작은 잠금장치
- 장식: 희귀 원석 전시 슬롯, 프로젝트 명패, 배지 진열판
- 이펙트: 잠금 해제 빛, 보안 스캔 라인

### 11-5. 광부 / 장비 레이어 예시
- 헤드: 헤어, 안전모, 모자
- 얼굴: 안경, 표정, 얼굴 액세서리
- 상의: 작업복, 자켓, 상의 스킨
- 하의/신발: 작업 바지, 부츠
- 장비: 곡괭이, 손전등, 배낭, 장갑
- 장식: 배지 프레임, 칭호 리본
- 이펙트: 곡괭이 잔상, 희귀 광맥 발견 모션

## 12. 메타 루프 도입 순서
경제/꾸미기 구조는 한 번에 모두 열지 않고 단계적으로 도입하는 것이 적절하다.

### 12-1. Phase 1
- 본격 오픈하지 않거나 티저 수준으로만 예고
- 메인 제품 루프 방해 금지

### 12-2. Phase 2
- 핑크 결정 오픈
- 꾸미기 상점 오픈
- 작업실 / 광부 기본 꾸미기 도입
- 단순한 공간 성장 감각 검증

### 12-3. Phase 3
- 실험실 / 금고 꾸미기 확장
- 고대 동전 도입
- 고대 교환소 오픈
- 희귀 장식 / 시즌 장식 / 반응 요소 강화

### 12-4. Phase 4 이후
- 가벼운 케어 루프 강화
- 시즌 상점 / 이벤트 교환 구조 확장
- 희귀 수집도감 / 테마룸 / 추가 캐릭터 시스템 검토

## 13. 경제 밸런스 원칙
구체 수치는 별도 밸런스 실험 문서에서 다루되,
현재 문서에서는 방향 원칙만 고정한다.

### 13-1. 핑크 결정
- 꾸준히 하면 모이는 재화
- 활동량이 많을수록 더 많이 벌게 한다
- 소비 빈도가 높다

### 13-2. Pink Diamond
- 희귀하고 아껴 쓰는 재화
- 주된 공급원은 결제
- 무료 보상으로도 아주 소량은 제공 가능

### 13-3. 고대 동전
- 낮은 확률 드롭
- 이벤트 / 특별 광맥과 잘 연결됨
- 희귀 교환용으로만 사용

### 13-4. 철학 요약
- **핑크 결정** = 열심히 캐면 내 공간이 자란다
- **Pink Diamond** = 내 광산을 더 특별하게 만든다
- **고대 동전** = 드물게 발견하는 특별한 보상이다

## 14. 주의점
### 14-1. 메타가 본체를 잡아먹지 않게 할 것
아이디어 채굴과 문서화가 본체이며,
꾸미기와 케어는 보조 루프여야 한다.

### 14-2. 패널티형 케어 금지
밥을 안 줬다고 기능이 약해지거나,
매일 챙기지 않으면 손해를 보는 구조는 피한다.

### 14-3. 재화 종류 과도 증가 금지
초기 단계에서 재화를 과하게 늘리면 이해 난도가 높아지고,
기획 툴로서의 명확성이 무너질 수 있다.

### 14-4. 희귀 보상은 희귀해야 한다
고대 동전이 너무 흔하면 의미가 약해지고,
너무 안 나오면 기대감이 사라진다.

## 15. 정리
현재 기준에서 IDEA MINE의 게임 경제/꾸미기 구조는 아래처럼 정리한다.

- **핑크 결정** = 기본 꾸미기 재화
- **Pink Diamond** = 프리미엄 재화
- **고대 동전** = 랜덤 획득형 희귀 교환 재화

그리고 메타 공간은 아래 4축으로 확장한다.
- 작업실
- 실험실
- 금고
- 광부 / 장비

이 구조는
- 역할이 분명하고,
- 게임성과 기획 툴성을 동시에 해치지 않으며,
- 이후 밸런싱과 수익화 실험에도 유연하게 대응할 수 있다.
