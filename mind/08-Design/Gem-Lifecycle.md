# Gem Lifecycle

> 원석(아이디어)이 채굴부터 전시까지 거치는 시각적 변화 단계를 정의한다.
> 각 단계는 세계관 상의 행위에 대응하며, 희귀도와 교차하여 비주얼 조합을 만든다.

---

## 전체 여정

```
[광산]        [금고]          [실험실]         [실험실]         [전시장]
 채굴    →    보관     →     개요서 정제  →   MVP 설계    →   외부 공개

 Raw         Vaulted        Refined         Crafted        Showcased
 거친 돌      다듬은 돌       빛나는 보석      설계된 보석      전시된 보석
```

---

## 시각 단계 (5단계)

### Stage 1: Raw (원석)

- **시점**: 채굴 직후, 결과 화면
- **형태**: 거친 사각형 (borderRadius: 0)
- **색감**: 어둡고 불투명 (회색 계열)
- **빛**: 없음
- **느낌**: 방금 캔 돌멩이. 뭔지 아직 모르는 상태.

```
 ┌──────┐
 │▓▓▓▓▓▓│
 │▓▓▓▓▓▓│
 └──────┘
```

### Stage 2: Vaulted (보관 원석)

- **시점**: 금고에 반입 완료
- **형태**: 약간 다듬어진 형태 (borderRadius: 8, 모서리 깎임)
- **색감**: 밝아짐 (희귀도에 따른 색상 시작)
- **빛**: 은은한 glow
- **느낌**: 소중하게 보관된 원석. 가능성이 보이는 상태.

```
 ╭──────╮
 │░░░░░░│
 │░░░░░░│  + 은은한 glow
 ╰──────╯
```

### Stage 3: Refined (정제 원석 — 개요서 완료)

- **시점**: 실험실에서 개요서 생성 완료
- **형태**: 다이아몬드형 (45도 회전 + 축소)
- **색감**: 선명하고 빛남 (희귀도별 고유색)
- **빛**: 강한 glow + 스파클(✨)
- **느낌**: 정체가 밝혀진 빛나는 보석. "이 아이디어가 뭔지 안다."

```
    ◇
   ╱  ╲
  ╱ ✨ ╲   + 강한 glow
   ╲  ╱
    ◇
```

### Stage 4: Crafted (설계 원석 — MVP 청사진 완료)

- **시점**: Phase 3에서 MVP 청사진 + 로드맵 생성 완료
- **형태**: 보석이 설계도 프레임 안에 세팅됨
- **색감**: Refined와 동일한 보석 + 설계도 테두리(lab.bench 색상)
- **빛**: 보석 glow 유지 + 프레임에 미세한 하이라이트
- **느낌**: "이 아이디어를 어떻게 만들지까지 설계했다."

```
 ┌─────────┐
 │   ◆ ✨  │
 │ ═══════ │  설계도 라인
 │ ─── ─── │
 │ ─── ─── │
 └─────────┘
```

**개요서(Refined)와 청사진(Crafted)의 차이:**
- 개요서 = What (이 아이디어가 뭔지 정리)
- 청사진 = How (이걸 어떻게 만들지 설계)
- 보석 자체는 같고, **프레임(설계도)이 추가**되는 것

### Stage 5: Showcased (전시 원석 — Phase 3+)

- **시점**: Phase 3+ 쇼케이스에 공개
- **형태**: 보석이 받침대/왕관 위에 올라감
- **색감**: Refined/Crafted와 동일 + 골드 받침대
- **빛**: 강한 glow + 스포트라이트 효과
- **느낌**: "세상에 공개된 보석."

```
      ✨
    ◆ ✨
   ╱    ╲
  ┃ 받침대 ┃  + 스포트라이트
  ┗━━━━━━━┛
```

---

## 희귀도 (4종)

| 희귀도 | 광맥 | 색상 계열 | glow 색상 |
|--------|------|----------|----------|
| Common (일반) | 일반 광맥 | 하늘색/실버 | rgba(100,150,255,0.15) |
| Rare (희귀) | 희귀 광맥 | 보라/퍼플 | rgba(139,92,246,0.15) |
| Golden (금빛) | 금빛 광맥 | 골드/앰버 | rgba(196,176,122,0.15) |
| Legend (전설) | 전설 광맥 | 홀로그램/무지개 | rgba(255,255,255,0.15) |

색상 위계: 실버 → 보라 → 골드 → 무지개 (WoW, 디아블로, 원신 패턴)

## 시각 조합 매트릭스

```
              Common    Rare      Golden    Legend
Raw            ○         ○         ○         ○        → 4종
Vaulted        ●         ●         ●         ●        → 4종
Refined        ◆         ◆         ◆         ◆        → 4종
Crafted        ◆+프레임   ◆+프레임   ◆+프레임   ◆+프레임   → 4종
                                                      ───────
                                            Phase 1~3: 16종

Showcased      ◆+받침대   ◆+받침대   ◆+받침대   ◆+받침대   → 4종
                                            Phase 3+: +4종
                                            전체:     20종
```

---

## Phase별 필요 범위

| Phase | 필요 단계 | 비주얼 수 |
|-------|----------|----------|
| Phase 1 (S1~S3) | Raw, Vaulted, Refined | 12종 |
| Phase 3 (S10) | + Crafted | +4종 = 16종 |
| Phase 3+ | + Showcased | +4종 = 20종 |

---

## 상태 전환과 연출

| 전환 | 트리거 | 연출 |
|------|--------|------|
| Raw → Vaulted | 금고 반입 버튼 | 광차 출발 (MinecartDepart) |
| Vaulted → Refined | 개요서 생성 완료 | 원석 연마 (LabLoader Phase A) |
| Refined → Crafted | MVP 청사진 완료 | 보석이 설계도에 세팅되는 연출 (Phase 3) |
| Crafted → Showcased | 쇼케이스 공개 | 받침대에 올라가는 연출 (Phase 3+) |

---

## 감정(Appraisal) 표현

감정은 별도 시각 단계가 아니라, **Vaulted/Refined 보석 위에 배지로 표시**한다.

```
 ╭──────╮
 │░░░░░░│ [🔍 감정 완료]    ← Vaulted + 감정 배지
 ╰──────╯

    ◆ ✨  [🔍 감정 완료]    ← Refined + 감정 배지
```

감정은 "보석의 가치를 평가한 것"이지, 보석 자체를 변형시키는 게 아니기 때문.
