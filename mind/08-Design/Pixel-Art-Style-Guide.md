# Pixel Art Style Guide

> 픽셀 아트 에셋 제작 시 따라야 할 스타일 가이드.
> 구체적인 에셋 목록, 제작 파이프라인, 화면별 사양은 `plans/2026-03-23-pixel-asset-plan.md` 참조.

---

## 스타일 방향

- 해상도: **32x32px 베이스** (로우 픽셀), 앱에서 3~4배 확대
- 팔레트: theme.ts 컬러 토큰 기준 무드 가이드 (엄격한 색 수 제한 없음)
- 제작: SeaArt 레퍼런스 생성 -> PixelLab 픽셀 전환
- 참고 스타일: 스타듀밸리, 하데스, 셀레스트 등에서 영감

---

## 필요 에셋 카테고리

### 캐릭터

- 광부 기본 캐릭터
- 표정/감정 변화
- 장비 착용 변형 (안전모, 작업복 등)

### 공간

- 광산(홈) 배경
- 실험실 배경
- 금고 배경
- 작업실 배경

### 아이템

- 원석 (일반, 반짝, 희귀)
- 곡괭이
- 핑크 결정
- Pink Diamond
- 고대 동전

### UI 요소

- 버튼
- 카드 프레임
- 아이콘 세트
- 배지

---

## 제작 원칙

- 크로스 플랫폼 일관성: 앱/웹에서 동일하게 보여야 함
- 스케일링: 정수 배율로만 확대 (픽셀 깨짐 방지)
- 애니메이션: 스프라이트 시트 기반
