# Camp 화면 레이아웃

> Phase 0 레이아웃 잡기. 데이터는 전부 더미/플레이스홀더.

## 결정 사항

- **프로필 카드 중심 레이아웃** + 캐릭터 영역 상단 독립
- Camp = "내 공간" → 나중에 꾸미기 화면으로 발전 (Phase 2)
- 캐릭터 위, 정보 아래 구조 → 시각적 영역 / 정보 영역 분리
- 데이터 전부 더미 (DB 스키마 미완)
- 설정 화면은 이번에 안 만듦 (메뉴 항목만)

## 레이아웃

```

│  ╔═══════════════════════════════╗   │
│  ║  광부                   Lv.1 ║   │  ← PixelCard (gold)
│  ║  email@email.com             ║   │
│  ║                              ║   │
│  ║  [ 🏅 ] [ 🏅 ] [ 🏅 ]      ║   │  ← 배지 슬롯 x3 (빈 원형)

│                                     │
│  ┌───────────────────────────────┐  │
│  │  채굴 기록                     │  │  ← PixelCard (default)

│                                     │
│  ┌───────────────────────────────┐  │
```

## 컴포넌트 매핑

| 영역 | 컴포넌트 | 비고 |
|------|----------|------|
| 타이틀 | PixelText variant="title" | "My Camp" |
| 캐릭터 | View (placeholder) | 64x64 bg.surface, ⛏️ 이모지 중앙 |
| 프로필 카드 | PixelCard variant="gold" | 이름+레벨+이메일+배지 슬롯 |
| 배지 슬롯 | View x3 | 24x24 원형, border.default, 빈 상태 |
| 채굴 기록 | PixelCard variant="default" | 3개 스탯 가로 배치 |
| 메뉴 리스트 | Pressable 리스트 | 설정만 터치 가능, 나머지 잠금 |
| 로그아웃 | PixelButton variant="danger" | 기존 기능 유지 |

## 더미 데이터

```
이름: "광부"
레벨: 1
이메일: session.user.email (실제)
배지: 빈 슬롯 x3
연속 채굴: 0일
총 채굴: 0회
보유 원석: 0개
```

## 확장 포인트

- 캐릭터 영역 → Phase 2: PixelImage 아바타 + 작업실 배경
- 배지 슬롯 → Phase 2: 실제 배지 아이콘
- 채굴 기록 → Phase 1: Supabase에서 실제 데이터
- 설정 메뉴 → Phase 1: camp/settings 화면 연결
- 구독/배지 메뉴 → Phase 1.5/2: 잠금 해제
