# Phase 1 MVP

> 핵심 루프 검증. 스플래시 -> 온보딩 -> 홈(광산) -> 광맥 선택 -> 원석 10개 -> 금고 반입 -> 실험실 -> 프로젝트 개요 -> 감정.

---

## MVP 목표

가장 먼저 검증해야 하는 것: **사람들이 이 조합형 아이디어 생성 경험을 재미있고 유용하다고 느끼는가?**

---

## 포함 화면

- 스플래시 / 진입 화면
- 온보딩 화면
- 홈 / 광산 화면 (The Mine)
- 원석 결과 화면
- 금고 화면 (The Vault)
- 실험실 화면 (The Lab)
- 요금제 / 업그레이드 화면
- 마이 / 프로필 화면 (My Mine)

---

## MVP 메뉴 구조 (4개)

1. **홈/광산 (The Mine):** 오늘의 광맥, 리롤, 퀘스트
2. **실험실 (The Lab):** 커스텀 조합, 개요서 생성
3. **금고 (The Vault):** 저장, 히스토리
4. **광부수첩 (My Mine):** 프로필, 재화, 설정

---

## 포함 기능

- 오늘의 광맥 3개 노출
- 다시 파기(리롤) 2회
- 광맥 기반 원석 10개 생성
- 금고 반입 (반입 슬롯 시스템)
- 프로젝트 개요 생성
- 기본 감정 생성
- 짧은 방향 정리 제공
- 광부 레벨 / 반입 슬롯 / 연속 채굴
- 데일리 퀘스트 기본형
- 광고로 1회 추가
- Lite / Pro 차이 노출
- Clean Mine Protocol 기본 1~3단계
- 배지 3종 (꾸준한 광부, 정제 완료 기획자, 실험실 숙련자)

- 원석 비교 / 분기 / 재정제
- MVP 청사진 full / Phase 로드맵 full
- 고급 실험실 모드
- 팀 협업
- 폴더 / 컬렉션 고도화

## 티저 기능 (공사 중 연출)

- MVP 청사진 미리보기
- 취향 광맥 해금 예고
- 쇼케이스 / 거래소 / 전망대 공사 중

## 기술 스택

- Supabase (Auth + DB + Storage)
- Python (비즈니스 로직)
- OpenAI API (아이디어 생성)
- RevenueCat (모바일 결제) + Polar.sh (웹 결제)
