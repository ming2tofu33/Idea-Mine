# Currency System (재화 구조)

> 2통화 + 희귀 재화 + 성장 자산 구조.

---

## 재화 철학

- **핑크 결정** = 열심히 캐면 내 공간이 자란다
- **Pink Diamond** = 내 광산을 더 특별하게 만든다
- **고대 동전** = 드물게 발견하는 특별한 보상이다

---

## 1. 핑크 결정 (기본 재화)

노력형 / 축적형 / 일상형 재화.

**획득처:**
- 광맥 채굴 완료
- 아이디어 원석 금고 반입
- 프로젝트 개요 생성
- 감정 완료
- 데일리 퀘스트 완료
- 연속 채굴 보상
- 광부 레벨업 보상
- 배지 획득 보상

**사용처:**
- 작업실/광산/실험실/금고 꾸미기
- 광부 캐릭터 기본 꾸미기
- 가벼운 케어 아이템

---

## 2. Pink Diamond (프리미엄 재화)

프리미엄형 / 희귀형 / 선택형 재화.

**공급:** 인앱 결제 중심, 무료 보상 아주 소량.

**사용처:**
- 희귀 가구, 한정 캐릭터 스킨, 시즌 장식
- 꾸미기 슬롯 확장, 금고 확장
- 기능 확장권 (야간 작업권, 정밀 감정권, 청사진 확장권)

---

## 3. 고대 동전 (랜덤 희귀 재화)

랜덤 보상형 / 이벤트형 / 희귀 교환 재화.

**획득:** 채굴 중 낮은 확률 드롭, 주말 광맥 보상, 이벤트.

**사용:** 고대 교환소 전용 (희귀 장식, 한정 스킨, 시즌 아이템).

---

## 4. 반입 슬롯 (성장 자산)

통화가 아니라 성장 자산. 레벨업에 따라 확장.

---

## 밸런스 원칙

- 핑크 결정: 꾸준히 하면 모이는 재화, 소비 빈도 높음
- Pink Diamond: 희귀하고 아껴 쓰는 재화, 결제 중심
- 고대 동전: 낮은 확률 드롭, 희귀 교환용으로만 사용
- 재화 종류 과도 증가 금지
