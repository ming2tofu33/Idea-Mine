# Meta Loop (메타 보상 루프)

> 본체 루프에 보상을 붙여 재방문성과 애착을 강화하는 보조 루프.

---

## 루프 구조

```
본체 루프: 광맥 -> 아이디어 원석 -> 금고 -> 실험실
메타 루프: 채굴/문서화 보상 -> 재화 획득 -> 공간/캐릭터 성장
```

---

## 메타 루프가 보조해야 할 4가지 감정

### 1. 축적의 재미
채굴하고 문서화할수록 내 광산과 작업 공간이 자란다는 감각.

### 2. 애착의 재미
내 캐릭터, 내 작업실, 내 실험실이라는 감각.

### 3. 발견의 재미
가끔 특별한 보상이 나오고, 희귀한 장식과 재화를 발견하는 재미.

### 4. 돌봄의 재미
가벼운 케어 요소를 통해 캐릭터와 공간에 애착을 붙이는 재미.
단, 패널티형 의무 관리로 가지 않는다.

---

## 핵심 원칙

- 메타가 본체를 잡아먹지 않게 할 것
- 패널티형 케어 금지 (밥 안 줬다고 기능 약해지는 구조 X)
- 희귀 보상은 희귀해야 한다

---

## Phase별 도입 순서

| Phase | 범위 |
|-------|------|
| Phase 1 | 티저 수준만, 본체 루프 방해 금지 |
| Phase 2 | 핑크 결정 + 꾸미기 상점 + 기본 꾸미기 |
| Phase 3 | 고대 동전 + 고대 교환소 + 확장 꾸미기 |
| Phase 4+ | 케어 루프 강화, 시즌 상점, 수집 도감 |
