# Project Vision

> IDEA MINE은 AI, 대상, 산업, 형태, 가치, 수익모델을 조합해 기획 아이디어를 발굴하고 발전시키는 픽셀 아트 기반 아이디어 제너레이션 플랫폼이다.

---

## 한 줄 정의

> 픽셀 광산 세계관 속에서 AI 서비스 아이디어를 가볍게 채굴하고, 마음에 드는 결과를 금고에 쌓아가는 게임형 기획 도구

---

## 문제 정의

AI 기술은 빠르게 늘어나고 있지만, 기획자는 다음 문제를 반복해서 겪는다.

- 어떤 기술을 어떤 타겟에게 붙여야 할지 감이 잘 오지 않는다
- 아이디어가 있어도 비즈니스 가치와 수익 모델까지 연결하기 어렵다
- 떠오른 아이디어가 일회성 메모로 끝나고 자산화되지 않는다
- 좋은 조합을 찾기까지 시간이 오래 걸리고, 실패한 시도는 기록되지 않는다

---

## 해결 방식

아이디어 발상을 **광산 채굴 메타포**로 단순화한다.

| 활동 | 메타포 |
|------|--------|
| 키워드 탐색 | 채굴 |
| 조합 실험 | 제련 |
| 개요서 발전 | 가공 |
| 공개 전시 | 쇼케이스 |
| 투자자 연결 | 거래소/매칭 |

---

## 핵심 가치 제안

- **빠른 발상:** 빈 화면에서 시작하지 않게 해준다
- **구조화된 기획:** AI, Who, Domain, Tech, Value, Money를 한 번에 보게 한다
- **아이디어 자산화:** 생성 -> 개요서화 -> 보관 -> 전시까지 이어진다
- **차별적 경험:** 픽셀 아트와 광산 세계관으로 감성적 몰입감을 만든다

---

## 제품 방향성

> "일처럼 쓰는 게임, 게임처럼 들어오는 기획 툴"

- 게임처럼 진입 장벽이 낮다
- 실무 툴처럼 결과물은 남는다

---

## 차별화 포인트

1. 단순 프롬프트 입력형이 아니라 **조합 구조가 명확함**
2. 아이디어 생성에서 끝나지 않고 **개요서화/전시/연결**로 이어짐
3. 광산 세계관과 픽셀 아트로 **브랜드 자산화가 쉬움**
4. 실패한 조합도 기록 가능한 **실험 아카이브**로 발전 가능
5. **MCP 서버**로 AI 코딩 도구(Claude Code, Cursor 등)에서 직접 체험 가능 — 마찰 제로 획득 채널

---

## 기술 스택

| 영역 | 선택 | 이유 |
|------|------|------|
| Frontend | Expo (React Native) + Expo Router | 모바일 앱 + 웹 단일 코드베이스 |
| Backend | Supabase (Auth + DB + Storage) | 이미 경험 있음, 1인 개발 최적 |
| Backend 로직 | Python | 이미 경험 있음 |
| AI | OpenAI API | 아이디어 생성 엔진 |
| 모바일 결제 | RevenueCat | Apple IAP + Google Play 래퍼 |
| 웹 결제 | Polar.sh | MoR, 세금/VAT 대행 |
| Ads | Google AdMob | 리워드 광고 |
| Push | Expo Notifications | 네이티브 푸시 |
| Build | EAS Build | 클라우드 빌드 (Windows에서 iOS 가능) |
| MCP 서버 | Python + FastMCP | 획득 채널. 백엔드 API 래퍼 |
| Domain | ideamineai.com | 확보 완료 |
