# Project Vision

> IDEA MINE은 AI를 활용해 아이디어 자산을 탐사하고, 조합하고, 정제하고, 축적하는 웹 우선 프리미엄 아이디어 탐사 플랫폼이다.

---

## 한 줄 정의

> 프리미엄 탐사 시스템 속에서 AI 서비스 아이디어를 발견하고, 선별하고, 기획 자산으로 축적해가는 웹 기반 창작 도구

---

## 문제 정의

AI 기술은 빠르게 늘어나고 있지만, 기획자는 다음 문제를 반복해서 겪는다.

- 어떤 기술을 어떤 타겟에게 붙여야 할지 감이 잘 오지 않는다
- 아이디어가 있어도 비즈니스 가치와 수익 모델까지 연결하기 어렵다
- 떠오른 아이디어가 일회성 메모로 끝나고 자산화되지 않는다
- 좋은 조합을 찾기까지 시간이 오래 걸리고, 실패한 시도는 기록되지 않는다

---

## 해결 방식

아이디어 발상을 **탐사와 축적의 구조**로 단순화한다.

| 활동 | 메타포 |
|------|--------|
| 키워드 탐색 | 채굴/발견 |
| 결과 선별 | 감정/판단 |
| 개요서 발전 | 정제/확장 |
| 아이디어 저장 | 자산 축적 |
| 공개 전시 | 쇼케이스 |
| 투자자 연결 | 거래소/매칭 |

---

## 핵심 가치 제안

- **빠른 발상:** 빈 화면에서 시작하지 않게 해준다
- **구조화된 기획:** AI, Who, Domain, Tech, Value, Money를 한 번에 보게 한다
- **아이디어 자산화:** 발견 → 선별 → 정제 → 축적까지 이어진다
- **바이브 코딩 입력물:** 풀 개요서를 AI 코딩 도구에 넣으면 바로 MVP를 만들 수 있다
- **차별적 경험:** 프리미엄 스페이스 UI로 진지한 창작 도구의 신뢰감을 만든다

---

## 제품 방향성 (v2)

> "진지한 창작 도구, 몰입감 있는 탐사 경험"

- 채팅창이 아니라 관측소와 연구 시스템에 가까운 경험
- 사용자는 탐사자이자 선별자이며 구축자
- Premium 먼저, Playful은 그 다음

### v2 핵심 전환

1. **앱 우선 → 웹 우선:** 읽기/비교/선별/정리는 브라우저가 더 적합
2. **픽셀 게임 UI → 프리미엄 스페이스 UI:** Cinematic Observatory 방향
3. **기능 폭 → 핵심 루프 강화:** Mine-Vault-Lab 루프를 먼저 단단하게

---

## 차별화 포인트

1. 단순 프롬프트 입력형이 아니라 **조합 구조가 명확함**
2. 아이디어 생성에서 끝나지 않고 **정제/축적/전시/연결**로 이어짐
3. 탐사 세계관과 프리미엄 UI로 **브랜드 자산화가 쉬움**
4. 실패한 조합도 기록 가능한 **실험 아카이브**로 발전 가능

---

## 기술 스택 (v2)

| 영역 | 선택 | 이유 |
|------|------|------|
| Frontend (Web) | React + Next.js + TypeScript | 웹 우선 제품, 강한 컴포넌트 구조 |
| Motion | Framer Motion | 프리미엄 모션 계층 |
| Graphics | CSS + Canvas | 기본, 선택적으로 React Three Fiber |
| Backend | Supabase (Auth + DB + Storage) | 이미 경험 있음, 1인 개발 최적 |
| Backend 로직 | Python (FastAPI) | 이미 경험 있음 |
| AI | OpenAI API | 아이디어 생성 엔진 |
| 웹 결제 | Polar.sh | MoR, 세금/VAT 대행 |
| Domain | ideamineai.com | 확보 완료 |

---

## MVP 범위 (v2)

- 웹 진입 구조 + 로그인 흐름
- Mine: 탐사, 생성, 1차 선별
- Vault: 저장, 재방문
- Lab: 선택한 자산을 구조화된 기획 결과로 확장
- 프리미엄 스페이스 디자인 시스템 첫 버전
- 기존 백엔드/데이터 계층 연동
