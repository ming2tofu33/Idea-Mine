# Competitive Analysis

> IDEA MINE의 핵심 경쟁 위협 분석과 방어 전략. "ChatGPT에 물어보면 되잖아?"에 대한 구조적 답변.

---

## 핵심 경쟁 위협

### ChatGPT / 범용 LLM

가장 큰 위협. 사용자가 ChatGPT에게 "AI 기술 x 헬스케어 x 구독 모델로 사업 아이디어 10개 알려줘"라고 치면 무료로 비슷한 결과를 얻을 수 있음.

### 기존 AI 아이디어 도구

| 경쟁사 | 특징 | IDEA MINE 대비 |
|--------|------|---------------|
| ChatGPT/Claude | 범용 LLM, 무료~저가 | 구조 없음, 저장 없음, 축적 없음 |
| Ideamap | 마인드맵 기반 아이디어 정리 | 생성이 아닌 정리 도구 |
| Bizway | AI 비즈니스 플랜 생성 | 전통적 SaaS UI, 게임 요소 없음 |
| Validator AI | 아이디어 검증 도구 | 생성이 아닌 검증에 초점 |
| Ideanote | 기업용 아이디어 관리 | B2B 중심, 개인 사용 부적합 |

## ChatGPT 대비 IDEA MINE 방어 포인트

### 1. 제약의 역설 (Constraint Paradox)

창의성 연구에서 반복적으로 증명된 사실: "빈 캔버스보다 제약이 있을 때 더 창의적인 결과가 나온다." (Stokes 2005, Creativity from Constraints)

ChatGPT에게 "AI 사업 아이디어 알려줘"라고 하면 너무 뻔하거나 너무 추상적인 결과가 나옴. 제약이 없으니까. IDEA MINE은 "음성 AI x 1인가구 x 헬스케어 x 앱 x 감정케어 x 구독"이라는 6차원 제약을 건 상태에서 생성. 이건 ChatGPT에게 장문의 프롬프트를 정교하게 쓰는 것보다 UX적으로 빠르고, 결과적으로 더 구체적인 아이디어를 만듦.

IDEO의 디자인 씽킹에서도 "How Might We" 질문에 제약을 거는 것이 핵심 기법. "어떻게 하면 더 나은 건강 앱을 만들까?"보다 "어떻게 하면 1인가구 시니어가 IoT로 수면을 추적하게 할까?"가 훨씬 실행 가능한 아이디어를 만든다. IDEA MINE의 6원석 조합은 이 방법론을 자동화한 것.

### 2. 세렌디피티 엔진 (Serendipity Engine)

ChatGPT는 사용자가 이미 아는 것을 물어봄. IDEA MINE은 사용자가 생각하지 못했을 조합을 제시함.

예시: "감정인식 AI x 디지털 노마드 x 리걸테크 x 웨어러블 x 심리케어 x 성공보수형"

이 조합을 스스로 떠올릴 기획자는 거의 없음. 이 "강제된 우연"이 IDEA MINE의 진짜 가치. ChatGPT에는 이 구조가 없음.

### 3. 구조적 차별점

| ChatGPT | IDEA MINE |
|---------|-----------|
| 매번 프롬프트를 직접 작성해야 함 | 키워드 선택만 하면 조합이 완성됨 |
| 결과가 일회성, 저장/관리 안 됨 | 금고에 축적, 이력 관리, 비교 가능 |
| 아이디어만 나옴 | 아이디어 -> 개요서 -> 감정까지 파이프라인 |
| 조합의 체계가 없음 | 6가지 축이 구조화되어 빠짐없이 사고 |
| 재미가 없음 | 매일 새로운 광맥, 게이미피케이션 |
| 히스토리 추적 안 됨 | 채굴 DNA, 선호 패턴 분석 가능 |

### 4. MCP로 직접 체험시키기

"ChatGPT로 해도 되잖아?"에 대한 가장 직접적인 답: "저희 MCP를 깔아보세요."

MCP를 설치하면 Claude Code/Cursor 안에서 바로 IDEA MINE을 체험할 수 있음. ChatGPT에 프롬프트를 직접 작성하는 것보다 빠르고, 6차원 조합 구조의 가치를 텍스트로 즉시 느낄 수 있음. 상세: `Tier-Structure`, `MCP-Distribution-Channel` 참조.

### 5. 마케팅 전환: 비교 콘텐츠

"ChatGPT vs IDEA MINE" 비교 콘텐츠를 직접 만들어 방어 논리를 마케팅 자산으로 전환:

```
[트위터 스레드 예시]
"ChatGPT에게 'AI 사업 아이디어 10개'를 요청했다.
결과: 챗봇, 추천 엔진, 자동화... 누구나 떠올릴 것들.

같은 요청을 6가지 제약 조건과 함께 했다.
음성AI x 반려동물가구 x 슬립테크 x IoT x 건강트래킹 x 구독

결과: [스크린샷]
제약이 창의성을 만든다. 이게 IDEA MINE이 하는 일."
```

이 콘텐츠 하나가 경쟁 방어 + 가치 설명 + 바이럴 포텐셜을 동시에 해결.

## ChatGPT 사용자 전환 전략 (Status-Quo Bias 대응)

ChatGPT가 이미 "현재 도구"인 사용자를 IDEA MINE으로 전환시키려면, "왜 낫냐"를 설명하는 것만으로 부족. "어떻게 쉽게 전환하는가"까지 설계해야 함.

### 전환 마찰 최소화

1. **5분 무료 체험 (로그인 불필요):** 웹에서 즉시 조합 1회 체험 -> 결과를 보여준 후 "이 아이디어를 저장하려면 가입하세요" (Endowment: 이미 만든 것을 잃고 싶지 않음)
2. **ChatGPT 결과 가져오기:** "이미 ChatGPT에서 아이디어를 만들었나요? 텍스트를 붙여넣으면 IDEA MINE 형식으로 정리하고 키워드 DNA를 분석해드립니다" (역채굴 기능과 연결)
3. **비교 체험:** "같은 주제로 ChatGPT와 IDEA MINE을 동시에 써보세요" 가이드 제공

### 전환 포인트 설계

사용자가 ChatGPT에서 IDEA MINE으로 넘어오는 결정적 순간:
- "ChatGPT에서는 매번 프롬프트를 다시 써야 하지만, 여기선 클릭 3번이면 된다"
- "ChatGPT에서는 어제 만든 아이디어를 다시 찾을 수 없지만, 여기선 금고에 다 있다"
- "ChatGPT에서는 혼자 쓰지만, 여기선 내 채굴 패턴을 분석해준다"

### 타겟 랜딩: "ChatGPT로 아이디어를 만들고 계신가요?"

ideamineai.com/from-chatgpt 별도 랜딩:
```
"ChatGPT로 AI 사업 아이디어를 만들고 계신가요?
좋은 시작이에요. 하지만 이런 문제를 겪고 있을 겁니다:
- 매번 같은 프롬프트를 다시 써야 함
- 좋은 아이디어가 나왔는데 어디 저장했는지 모름
- 새로운 조합을 떠올리기 어려움

IDEA MINE은 이걸 해결합니다. [무료 체험]"
```

## 방어 가능한 해자(Moat) 후보

### 단기 해자

- **세계관 + 비주얼 브랜딩**: 픽셀 아트 광산 테마는 쉽게 복제 불가
- **큐레이팅된 키워드 택소노미**: 109개 키워드가 AI 프롬프트에 최적화되어 있음
- **조합 구조 UX**: 6차원 제약 기반 생성은 ChatGPT의 빈 프롬프트와 근본적으로 다른 경험

### 중기 해자

- **사용자 데이터 축적**: 채굴 히스토리, 저장 패턴, 선호 조합 -> 개인화
- **콘텐츠 네트워크 이펙트**: 사용자가 만든 아이디어가 다른 사용자에게 가치
- **커뮤니티 키워드 확장**: 사용자 제안 키워드가 택소노미를 풍부하게 만듦

### 장기 해자

- **플랫폼 이펙트**: 쇼케이스/거래소가 열리면 양면 마켓플레이스
- **Switching Cost**: 금고에 쌓인 아이디어 포트폴리오는 다른 도구로 이전 불가

## 포지셔닝 전략

### "구조화된 이노베이션 도구"로 포지셔닝

"아이디어 장난감"이 아니라 "디자인 씽킹의 How Might We를 자동화한 구조적 이노베이션 도구"로 프레이밍하면:
- B2B 워크숍/해커톤 유스케이스로 확장 가능
- AI PM/기획자 타겟에게 신뢰감 부여
- 학계/교육 기관 파트너십 가능성
