# Tier Structure

> Free Miner / Mine Owner Lite / Mine Owner Pro 3단계 티어. 체험 -> 반복 -> 실행의 명확한 경계.

---

## 티어 철학

### 한 줄 정의

- **Free (기본 광부):** 채굴하고, 원석 보고, 개요 만들고, 기본 감정까지 해보는 **체험형**
- **Lite (광산주 Lite):** 광고 없이 여러 번 채굴하고, 개요와 평가를 반복하는 **실속형**
- **Pro (광산주 Pro):** 평가를 넘어 실행 설계, 우선순위, 로드맵까지 만드는 **작업형**

### 핵심 원칙

- Free는 "가치 체험"까지 -- 제품 실력이 느껴져야 함
- Lite는 "반복 사용"까지 -- 가볍지만 충분한 플랜. **부족한 요금제로 보이면 안 됨**
- Pro는 "실행 도구"까지 -- 탐색 도구가 아니라 프로젝트 설계 작업실

### 경계선 설계: "깊이 레이어" 방식

같은 기능이 티어별로 출력의 깊이/상세도가 다름:
- Free: 기능 존재 확인 (1회 체험)
- Lite: 반복 사용 가능 + 심화 기능 얕은 버전
- Pro: 풀 퀄리티 + 실행 단계 기능

---

## 티어 이름

| 영문 | 한국어 | 정체성 |
|------|--------|--------|
| Free Miner | 기본 광부 | 체험하는 광부 |
| Mine Owner Lite | 광산주 Lite | 광산을 운영하기 시작한 사람 |
| Mine Owner Pro | 광산주 Pro | 광산을 본격 경영하는 사람 |

Free와 유료 사이에 "광부 -> 광산주"라는 정체성 격차를 만듦.

---

## AI 키워드 슬롯 (6번째 원석)

| 티어 | 원석 수 | AI 카테고리 | 방식 |
|------|---------|------------|------|
| Free | 5원석 | 잠금 | AI 슬롯 없음 |
| Lite | 6원석 | 해금, **랜덤 배정** | 시스템이 AI 키워드를 선택 (오늘의 광맥처럼) |
| Pro | 6원석 | 해금, **자유 선택** | 18개 AI 키워드 중 직접 고름 |

Lite도 6원석 품질을 경험하지만, "내가 원하는 AI 기술을 직접 고르는 정밀도"는 Pro에서만 가능. 세계관과도 일치: Lite 광산주는 "AI 광맥이 랜덤으로 뜸", Pro 광산주는 "AI 광맥을 직접 선택".

---

## 가격 가설 (Van Westendorp 결과로 조정 예정)

| 티어 | 월간 | 연간 | 연간 할인율 |
|------|------|------|-----------|
| Free | $0 | $0 | - |
| Lite | ~$4.99 | ~$47.90 | ~20% |
| Pro | ~$12.99 | ~$124.70 | ~20% |

근거:
- Lite-Pro 가격 격차를 크게 설정 -- Lite는 "가성비 선택"으로 많이 팔림, Pro는 "진짜 필요한 사람만"
- 경쟁사(Bizway $29, Validator AI $19)보다 낮은 개인용 가격대
- 연간 할인 20% -- 33% 이상은 월간 구독을 죽이므로 지양
- 이 숫자는 가설이며, Van Westendorp 4질문 결과로 최종 결정

## 기능별 티어 상세

### A. 홈 / 광산 (The Mine)

| 기능 | Free (기본 광부) | Lite (광산주 Lite) | Pro (광산주 Pro) |
|------|-----------------|-------------------|-----------------|
| 오늘의 광맥 | 3개 | 3개 | 3개 |
| 리롤 | 2회 | 넉넉 (10회) | 여유 (20회+) |
| 채굴 세션 | 1회/일 (광고 시청 시 +1회) | 5회/일 | 사실상 무제한 (일일 50회 소프트캡) |
| 광고 추가 채굴 | 1회 (보상형 광고) | 광고 없음 | 광고 없음 |
| 주말 특별 광맥 | 기본 참여 | 우선 접근 일부 | 우선 접근 + 희귀 광맥 |
| 취향 광맥 | 기본 | 해금 체감 향상 | 정밀도 높음 |
| 실험적 광맥 | - | 일부 노출 | 더 많이 노출 |
| AI 키워드 | 잠금 (5원석) | 랜덤 배정 (6원석) | 자유 선택 (6원석) |
| 아이디어당 키워드 사용 | 3~5개/5개 중 선택 | 3~5개/6개 중 선택 (AI 고정) | 3~5개/6개 중 선택 (AI 고정) |
| **풀 조합 모드** | - | - | **6개 전부 사용 (토글)** |

**포인트:**
- 무료도 재밌어야 하지만, 많이 캐는 즐거움은 Lite 이상에서 열린다.
- 유료 AI 키워드는 항상 포함 (빠지지 않음). 나머지 5개 중 2~4개가 선택됨.
- 각 아이디어 카드에 사용된 키워드가 태그로 표시됨.
- Pro의 풀 조합 모드: "모든 키워드 사용" 토글 시 10개 모두 6개 키워드 전부 반영. 기본 모드에서 방향을 찾고 -> 풀 조합으로 정밀하게 파는 2단계 채굴.

### B. 금고 (The Vault)

| 기능 | Free | Lite | Pro |
|------|------|------|-----|
| 저장 | 가능 (소용량) | 크게 증가 | 대용량 |
| 즐겨찾기/핀 | 가능 | 여유 | 여유 |
| 간단 메모 | 가능 | 편의성 증가 | 고급 메모 |
| 실험실로 보내기 | 가능 | 가능 | 가능 |
| 정리 기능 | 기본 | 정리 편의성 향상 | 컬렉션/태그 고급 |
| 비교/분기/재정제 | - | - | 가능 |
| 감정 이력 누적 | - | - | 발전 이력 관리 |

**금고 경계:** Free = 보관, Lite = 반복 관리, Pro = 작업 자산 관리

### C. 실험실 (The Lab)

이 공간이 **티어 차이를 가장 강하게 보여주는 곳**.

| 기능 | Free | Lite | Pro |
|------|------|------|-----|
| 프로젝트 개요 | 1회/일 | 여러 번 | 거의 무제한 |
| 기본 감정 | 1회 | 넉넉 | 충분 |
| 정밀 감정 | - | **얕은 버전** (점수 + 한 줄) | **풀 버전** (근거 + 사례 + 리스크) |
| 심층 감정 리포트 | - | - | 가능 |
| 방향 조언 | 아주 짧은 | 간단한 방향 정제 | 상세 방향 |
| 실행 설계 | - | **맛보기** (요약 3줄) | **풀 버전** (기능 목록 + 기술 스택) |
| MVP 청사진 | - | - | 가능 |
| Phase 로드맵 | - | - | 가능 |
| 비교/분기/재정제 | - | - | 가능 |
| 대안 시나리오 | - | - | 가능 |
| 커스텀 조합 (기존 키워드) | - | 일부 허용 | 완전 개방 |
| **커스텀 키워드 직접 입력** | - | **5카테고리, 10개 저장** | **6카테고리(AI 포함), 무제한** |
| AI 보정/추천 | - | 기본 (카테고리 적합성 + 보정) | 동의어 확장 + 잘 맞는 조합 추천 |

**깊이 레이어 예시:**

정밀 감정 (점수 없음, 코멘트만):
- Lite: "시장성 — 헬스케어 AI 시장이 성장 중이지만 경쟁이 치열해 진입 타이밍이 중요함"
- Pro: "시장성 — 헬스케어 AI 시장이 연 23% 성장 중. 유사 서비스 A사($2M 펀딩, 음성 특화), B사(MAU 5만, 범용). 이 아이디어의 음성 AI 특화는 A사와 직접 경쟁하지만, 1인가구 타겟은 아직 비어 있음. 다만 의료 데이터 규제 변화가 리스크"

실행 설계:
- Lite: "이 아이디어는 모바일 앱 형태가 적합하며, MVP는 핵심 기능 3개로 시작하는 것을 추천합니다"
- Pro: "MVP 기능 목록: (1) 음성 감정 입력 (2) AI 분석 리포트 (3) 주간 트래킹. 기술 스택 제안: React Native + Whisper API + Supabase. Phase 1 범위: 4주, Phase 2: 감정 트렌드 대시보드"

### D. 캠프 (Camp)

| 기능 | Free | Lite | Pro |
|------|------|------|-----|
| 프로필 | 기본 | 기본 | 기본 |
| 채굴 통계 | 기본 | 상세 | 상세 + 트렌드 |
| 아이디어 DNA 리포트 | - | 기본 분석 | 정밀 분석 |
| 광부 레벨 | 가능 | 가능 | 가능 |

## 커스텀 키워드 입력 시스템

### 입력 방식: 카테고리 슬롯 내 교체

6슬롯 구조를 유지하면서, 특정 슬롯의 키워드를 직접 타이핑으로 교체.

### 입력 흐름

```
사용자가 Domain 슬롯 탭
-> 기존 키워드 목록 (헬스케어, 에듀테크...) + 하단에 "직접 입력"
-> 텍스트 필드 + 안내: "이 슬롯은 '어떤 산업인가'를 정의해요"
-> 사용자 입력: "펫"
-> AI 판단:
   적합 -> "펫케어 산업으로 등록할게요" (자동 보정)
   모호 -> "혹시 이런 뜻인가요?" (펫케어 / 펫테크 / 반려동물 커머스)
   카테고리 불일치 -> "이 키워드는 '타겟층(Who)'에 더 어울려요. 이동할까요?"
```

### AI 보정 구현

- 1단계: 임베딩 유사도 검색 (LLM 호출 없음, 거의 무료) -- 90% 케이스 처리
- 2단계: 유사도가 낮을 때만 GPT-4o-mini 호출 (분류 + 보정)
- 비용: 유저 1,000명 기준 월 $3-24 수준

### Lite vs Pro 차이

| | Lite | Pro |
|--|------|-----|
| 직접 입력 카테고리 | Who, Domain, Tech, Value, Money (5개) | 전체 6개 (AI 포함) |
| 저장 가능 커스텀 키워드 | 10개 | 무제한 |
| AI 보정/추천 | 기본 (카테고리 적합성 + 보정) | 동의어 확장 + "이 키워드와 잘 맞는 조합" 추천 |

## 무료 -> Lite -> Pro 경계선 요약

### 무료에서 반드시 보여줘야 하는 것 (제품 실력 증명)

이건 안 보여주면 안 됨:
- 원석 10개 생성
- 프로젝트 개요 문서 1회
- 기본 감정 1회

이 3개는 무료 핵심 체험. 여기까지 못 보면 제품 실력이 안 느껴짐.

### Lite에서 확실히 열어줘야 하는 것 (돈값)

Lite가 "실속형"이 되려면:
- 광고 제거
- AI 키워드 6원석 (랜덤 배정)
- 개요 문서 반복 생성
- 기본 감정 넉넉
- 정밀 감정 얕은 버전
- 금고 여유
- 실험실 이용 편의성
- 커스텀 조합 일부

Lite는 "아이디어를 꽤 자주 굴리는 사람"이 편하게 써야 함.

### Pro에서만 확실히 가져가야 하는 것 (핵심 차별)

Pro를 사는 이유:
- AI 키워드 자유 선택 (18개 전체)
- 정밀 감정 풀 버전 (근거 + 사례 + 리스크)
- 심층 감정 리포트
- 실행 설계 풀 버전
- MVP 청사진
- Phase 로드맵
- 비교 / 분기 / 재정제
- 대안 시나리오

Pro는 "아이디어를 프로젝트로 내리는 권한"이 핵심.

## Monthly Discount Progress (충성 보상)

오래 구독할수록 보상이 커지는 진행도 시스템. 해지 시 진행도 완전 리셋.
UI: 프로그레스 바(1/4 → 4/4) + 다음 달 혜택 미리 표시.

### Pro — 할인 진행도 + 독점 기능

| 월차 | 가격 | 충성 보상 |
|------|------|----------|
| 1개월 | $12.99 (정가) | — |
| 2개월 | $11.99 | 금빛/전설 광맥 확률 +2% |
| 3개월 | $10.99 | 역채굴 해금 (아이디어 → 키워드 분석) |
| 4개월~ | $9.99 (최저가 고정) | 채굴 DNA 리포트 해금 + 전설 광맥 확정권 월 1회 |

Pro 보상 핵심: "더 많이"가 아니라 **"다른 사람은 아직 못 쓰는 것"**. 독점감이 유지 이유.

### Lite — 기능 보너스 (할인 없음)

Lite는 가격($4.99)이 이미 저렴해서 할인 여지가 없음. 대신 기능 보너스로 보상.

| 월차 | 충성 보상 | 효과 |
|------|----------|------|
| 1개월 | — | — |
| 2개월 | 금고 용량 +10개 | 해지하면 저장 공간 줄어 → 해지 방어 |
| 3개월 | AI 키워드 자유 선택 월 2회 (평소엔 랜덤) | "직접 고르니까 다르네" → Pro 업셀 유도 |
| 4개월 | 정밀 감정 풀 월 1회 + 실행설계 풀 월 1회 | Pro 핵심 체험 → 강한 업셀 |

Lite 보상 핵심: 2~3개월은 **해지 방어**(잃고 싶지 않은 것), 4개월은 **Pro 업셀**(갖고 싶은 것).

### 해지 후 재가입

진행도 완전 리셋 — 처음부터 다시 시작. "해지하면 쌓은 것을 잃는다"로 Loss Aversion 극대화.

## 무료 체험 / 보증 정책

- Pro 7일 무료 체험 (카드 등록 불필요) -- 써본 후 뺏기면 더 아까움 (Endowment Effect)
- 30일 환불 보증 -- 결정 후회 해소 (Regret Aversion)
- 해지 시 save offer: 다음 달 50% 할인 (1회)
