# Upsell Copy Guide

> 무료 한도 도달 / Lite -> Pro 업셀 시점의 인앱 문구 가이드. 톤은 압박형이 아니라 탐험 확장형.

---

## 업셀 톤 원칙

### 좋은 톤 (채택)

- 더 깊게 파기
- 더 정밀하게 보기
- 한 단계 더 다듬기
- 실제 프로젝트로 내리기
- 청사진까지 확장하기

### 피하는 톤 (금지)

- 지금 결제하세요
- 기능 잠금 해제
- 프리미엄 전용
- 업그레이드하지 않으면 불가

**핵심:** 압박형보다 탐험 확장형. "못 쓰게 막는다"가 아니라 "더 가고 싶게 만든다."

---

## Free -> 유료 업셀 문구

### A. 채굴 세션 소진

**느낌:** 오늘 할 만큼 했고, 더 캐려면 권한이 필요함

**문구 후보:**
- "오늘 채굴은 여기까지예요. 더 깊게 파려면 추가 채굴권을 열어보세요."
- "오늘 광산 작업이 끝났어요. 야간 작업권으로 한 번 더 이어갈 수 있어요."
- "오늘 채굴 가능한 광맥은 모두 확인했어요. 더 캐고 싶다면 광산주 권한이 필요해요."

**보상형 광고 버전:**
- "광고를 보고 야간 작업권을 받아 채굴을 1회 더 이어갈 수 있어요."
- "전광판 광고 시청 후 추가 채굴 1회를 열 수 있어요."

### B. 개요 문서 생성 한도 도달

**느낌:** 아이디어는 봤고, 진짜 정리된 문서는 더 가치 있는 결과물

**문구 후보:**
- "오늘 만들 수 있는 개요 문서는 모두 완성했어요."
- "이 원석을 문서로 더 발전시키려면 개요 확장권이 필요해요."
- "기본 채굴은 끝났어요. 이제 더 많은 아이디어를 문서로 다듬으려면 Mine Owner 플랜이 어울려요."

**보상형 광고 버전:**
- "광고 시청 후 개요 문서 1회 추가 생성이 가능해요."
- "이 원석을 놓치기 아쉽다면 정리권 1회를 열어보세요."

### C. 감정 심화 잠금

**느낌:** 표면은 봤고, 이제 더 정교한 판단이 필요함

**문구 후보:**
- "기본 감정은 완료됐어요. 이제 정밀 감정으로 강점과 리스크를 더 깊게 볼 수 있어요."
- "이 아이디어의 결을 더 자세히 보려면 심층 감정 리포트를 열어보세요."
- "지금은 표면만 확인했어요. 더 깊은 해석은 광산주 전용 도구에서 제공돼요."

### D. 실행 설계 잠금

**느낌:** 좋은 아이디어에서 실제 프로젝트로 넘어가는 분기점

**문구 후보:**
- "개요는 완성됐어요. 이제 MVP 청사진으로 실제 실행 순서를 잡아볼 차례예요."
- "이 아이디어를 프로젝트로 밀어붙이고 싶다면 실행 설계 확장이 필요해요."
- "지금은 방향을 본 단계예요. 다음은 Phase 설계로 내려가는 단계예요."

## Lite -> Pro 업셀 문구

Lite 유저는 이미 어느 정도 만족하고 있기 때문에, "더 많이"보다 **"더 깊게"**를 강조해야 함.

### 전환 포인트

- 정밀 감정을 자주 쓰는 유저 (얕은 버전의 한계 체감)
- 개요 문서를 여러 버전으로 실험하는 유저
- 원석을 비교/분기하고 싶은 유저
- 실행 설계 맛보기를 본 후 풀 버전이 궁금한 유저

### 문구 후보

- "아이디어를 모으는 단계는 충분히 하고 있어요. 이제 실행 가능한 설계 단계로 넘어가 볼까요?"
- "개요를 넘어 실제 프로젝트 구조까지 보고 싶다면 Pro가 더 잘 맞아요."
- "지금부터는 '좋은 아이디어'보다 '실행할 아이디어'를 골라야 할 단계예요."
- "자주 실험하고 문서까지 발전시키고 있다면, 이제 광산주 Pro가 더 자연스러워요."

### Lite vs Pro 차이 한 줄

```
Lite: 아이디어를 고르고 정리하는 플랜
Pro: 아이디어를 실행 가능한 프로젝트로 만드는 플랜
```

## 핵심 업셀 3포인트

처음부터 많지 않게, 이 3개만 잡아도 유료 구조가 선명해짐:

### 1) 추가 개요 문서 생성

"이 원석을 더 문서로 다듬고 싶다면 개요 확장권을 열어보세요."

### 2) 정밀 감정

"기본 감정은 끝났어요. 이제 정밀 감정으로 강점과 리스크를 더 자세히 볼 수 있어요."

### 3) MVP 청사진

"개요는 완성됐어요. 이제 MVP 청사진으로 실제 실행 순서를 잡아볼까요?"

## 각 티어가 사용자에게 주는 느낌

### Free (기본 광부)

> "하루에 한 번 들어와도 꽤 괜찮은 결과를 가져갈 수 있다."

문구 톤: 가볍고 친절. "오늘의 채굴", "한 번 더 캐기", "야간 작업권"

### Lite (광산주 Lite)

> "광고 없이 편하게, 생각날 때 충분히 써볼 수 있다."

강조 포인트:
- 광고 없음
- 하루 5회 안팎 넉넉한 사용
- 개요 문서와 기본 감정 중심
- 가끔 실무적으로 써도 안 막힘

문구 톤: "아이디어를 자주 정리하는 사람에게", "가볍지만 충분한 채굴 플랜", "생각날 때 막히지 않게"

### Pro (광산주 Pro)

> "이제 이건 놀이가 아니라 작업 도구다."

강조 포인트:
- 실행 설계 / MVP 청사진 / Phase 로드맵
- 비교/분기/재정제
- 반복 실험
- AI 키워드 자유 선택

문구 톤: "실제 프로젝트까지 밀어붙이는 사람에게", "실험실을 제대로 쓰는 플랜", "실행 가능한 수준까지 파고드는 플랜"
