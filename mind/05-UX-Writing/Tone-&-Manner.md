# Tone & Manner

> 단순한 생산성 툴이 아닌, 가볍게 시작하고 깊게 파는 느낌. 귀엽고 영리한 발상 보조자.

---

## 브랜드 키워드

- 픽셀 아트
- 핑크 포인트
- 레트로 게임성
- 영감 채굴
- 기획 실험실
- 팝핑 아이디어
- 데이터 기반 상상력

---

## 톤 원칙

- 차갑고 기술적인 AI보다 **귀엽고 영리한 발상 보조자** 느낌
- 말투는 너무 유치하지 않되, 살짝 게임적인 표현 사용
- 세계관 용어를 자연스럽게 섞되, 이해를 방해하면 안 됨

---

## 세계관 용어 가이드

| 일반 표현 | 세계관 표현 |
|----------|------------|
| 키워드 조합 세트 | 광맥 |
| 아이디어 | 원석 |
| 저장하기 | 금고에 반입 |
| 새로고침 | 다시 파기 / 리롤 |
| 개요서 만들기 | 제련 / 가공 |
| 평가하기 | 감정 |
| 공개하기 | 전시 |
| 프로필 | 광부수첩 |

---

## 사용 맥락별 톤 강도

| 맥락 | 세계관 톤 강도 | 예시 |
|------|--------------|------|
| 핵심 CTA | 중간 | "광맥을 선택하세요" |
| 보상/알림 | 강함 | "핑크 결정을 획득했어요" |
| 에러/경고 | 약함 | "저장에 실패했어요. 다시 시도해 주세요" |
| 설정/계정 | 없음 | "비밀번호를 변경합니다" |
| 결제/구독 | 약함 | "Pro 광부가 되어보세요" |

---

## 금지 사항

- 너무 딱딱한 생산성 툴 중심 표현 ("문서 작성", "보고서 생성")
- 과도한 게임 용어로 핵심 기능이 뭔지 모르게 되는 것
- 강제적/부정적 표현 ("놓치면 손해", "시간 없어요")
