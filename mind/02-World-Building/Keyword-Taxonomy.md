# Keyword Taxonomy

> 5개 키워드 카테고리가 하나의 광맥을 구성한다. 이것이 IDEA MINE의 핵심 도메인 모델.

---

## 5원석 체계

| 카테고리 | 질문 | 예시 |
|----------|------|------|
| **Who** | 누구를 위한가? | 1인가구, 시니어, 프리랜서, Z세대 |
| **Domain** | 어떤 산업인가? | 헬스케어, 교육, 금융, 커머스 |
| **Tech** | 어떤 기술을 쓰는가? | LLM, 컴퓨터비전, 음성AI, 멀티모달 |
| **Value** | 어떤 가치를 주는가? | 자동화, 감정케어, 비용절감, 접근성 |
| **Money** | 어떻게 돈을 버는가? | 구독, 프리미엄, 광고, B2B SaaS |

---

## 조합 방식

하나의 **광맥**은 5개 카테고리에서 각 1개씩 뽑은 키워드 세트다.

```
광맥 예시:
Who: 1인가구
Domain: 헬스케어
Tech: 음성AI
Value: 감정케어
Money: 구독
```

이 조합을 기반으로 AI가 아이디어 원석 10개를 생성한다.

---

## 키워드 운영 원칙

- 초기에는 카테고리당 15~20개 키워드로 시작
- 커스텀 키워드 추가는 Premium 기능
- 키워드는 트렌드에 따라 주기적으로 업데이트
- 사용자에게는 표/매트릭스가 아니라 **칩/원석/슬롯 UI**로 보여준다

---

## 아이디어 객체 구조

하나의 아이디어(원석)는 아래 정보를 가진다:

- 아이디어 제목
- 5가지 키워드 조합
- 3~4줄 요약
- 생성 시각
- 생성 방식 (랜덤/커스텀)
- 저장 여부
- 공개 여부
- 상세 개요서 여부

---

## 상세 개요서 객체 구조

- 문제 정의
- 타겟 유저
- 핵심 기능
- 차별점
- 시장 배경
- BM 구조
- 경쟁/대체재
- 초기 MVP 범위
- 리스크 및 가설
