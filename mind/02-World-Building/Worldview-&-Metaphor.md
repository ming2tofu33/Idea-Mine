# Worldview & Metaphor

> IDEA MINE의 모든 제품 경험은 '광산 채굴' 메타포 위에 구축된다.

---

## 메타포 매핑

| 실제 기획 활동 | 세계관 표현 | 설명 |
|---------------|------------|------|
| 키워드 조합 세트 | 광맥 (Vein) | 5~6개 키워드가 묶인 하나의 탐색 대상 |
| 생성된 아이디어 | 원석 (Raw Gem) | 광맥에서 채굴한 날것의 아이디어 |
| 아이디어 저장 | 금고 반입 (Vault) | 원석을 안전하게 보관 |
| 개요서 생성 | 제련/가공 (Refine) | 원석을 다듬어 프로젝트 개요로 발전 |
| 개요서 평가 | 감정 (Appraisal) | 아이디어의 시장성/실현성 평가 |
| 공개 전시 | 쇼케이스 (Showcase) | 다듬어진 결과를 외부에 공개 |
| 투자자 연결 | 거래소 (Exchange) | 아이디어와 투자자 매칭 |
| 리롤 | 다시 파기 | 새로운 광맥 탐색 |
| 키워드 카테고리 | 원석 유형 | AI, Who, Domain, Tech, Value, Money |

---

## 공간 체계

6개의 물리적 공간이 제품 IA를 구성한다.

| 공간 | 역할 | MVP 메뉴 |
|------|------|----------|
| The Mine | 키워드 발견, 채굴 | 홈 (메인) |
| The Lab | 키워드 조합, 실험 | 실험실 |
| The Vault | 아이디어/개요서 보관 | 금고 |
| The Showcase | 아이디어 공개 전시 | Phase 4 |
| The Exchange | 투자자-아이디어 매칭 | Phase 4 |
| The Observatory | 상위 지표/인사이트 | Phase 4 |

---

## 캐릭터/장비 체계

| 요소 | 세계관 표현 |
|------|------------|
| 사용자 | 광부 (Miner) |
| 사용자 레벨 | 광부 레벨 |
| 도구 | 곡괭이, 안전모, 장갑 |
| 저장 용량 | 반입 슬롯 |
| 업적 | 배지 |
| 기본 재화 | 핑크 결정 |
| 프리미엄 재화 | Pink Diamond |
| 희귀 재화 | 고대 동전 |

---

## 세계관 톤 원칙

- 세계관이 핵심 기능을 방해하면 안 된다
- 예쁘기만 하고 실제 기획 효율이 떨어지면 재방문이 약해진다
- 용어는 직관적이어야 한다 — 설명 없이도 무엇인지 짐작 가능해야 함
- 게임적 요소는 보조, 핵심은 결과물 품질
