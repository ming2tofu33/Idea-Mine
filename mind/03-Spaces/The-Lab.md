# The Lab (실험실 / 믹스)

> 직접 조합하고 실험하는 공간. 무료보다 프리미엄 가치를 느끼게 하는 핵심 공간.

---

## 공간 목표

- 사용자가 원하는 키워드를 직접 선택/조합
- 슬롯머신 형태의 재미있는 조합 경험
- AI 기반 아이디어 생성 및 비교
- Premium의 핵심 가치를 체감하는 공간

---

## 핵심 기능

### 키워드 커스텀 선택 (Premium)

- 5개 카테고리에서 원하는 키워드를 직접 선택
- 무제한 자유 조합

### 슬롯머신 조합

- 게임적 UI로 키워드 조합의 재미 강화
- 랜덤 + 부분 고정 조합 지원

### AI 아이디어 생성

- 조합 기반 아이디어 draft 10개 생성
- 결과 비교 및 선택

### 커스텀 키워드 관리 (Premium)

- 사용자가 직접 키워드 추가/관리
- 자신만의 관심 영역 반영

### 프로젝트 개요 생성

- 선택한 아이디어 1개에 대해 상세 개요서 생성
- 문제 정의, 타겟, 핵심 기능, BM 구조 등 포함

### 감정 (Appraisal)

- 생성된 개요서의 시장성/실현성 기본 평가
- 정밀 감정은 Phase 2에서 확장

---

## 실험실 문서 확장 흐름

프로젝트 개요 -> 감정 -> 방향 정리 -> MVP 청사진 -> Phase 로드맵

(MVP 청사진, Phase 로드맵은 Phase 3에서 full 개방)
