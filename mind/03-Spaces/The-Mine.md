# The Mine (광산)

> v2의 flagship 공간. 가장 cinematic하고 인상적. "들어오면 바로 한 판 시작"이 이 공간의 역할이다.

---

## 공간 목표

- 3초 안에 오늘 할 일이 보인다
- 30초 안에 첫 결과를 얻을 수 있다
- 다시 들어오고 싶을 이유가 남는다

---

## v2 분위기

- 가장 넓은 여백, 가장 깊은 장면 레이어링
- 가장 강한 글로우, 모션, 경이감
- 가장 강한 Signal Pink 강조
- 성운 깊이 레이어 + 미세 입자 배경

상세: `docs/Premium-Space-UI-Style-Guide.md` Section 13.1

## 핵심 기능

### 오늘의 광맥 3개

- 매일 랜덤 키워드 조합 3세트 제공
- 각 카드에 키워드 5~6개, 희귀도 표시
- API: GET `/mining/veins/today`

**희귀도 4단계:** Common(일반) → Rare(희귀) → Golden(금빛) → Legend(전설)

**희귀도 결정 기준:**
- Common: 기본 키워드 풀, 자연스러운 조합. 상시 등장.
- Rare: 기본 풀이지만 의외한 조합 (subtype 교차). 상시 등장.
- Golden: 분기 트렌드 키워드 1개 이상 포함. 분기 내내 등장.
- Legend: 시즌 한정 키워드 포함. 시즌 이벤트 기간에만 등장.

**등장 확률 (광맥 3개 각각 독립 적용):**

| 조건 | 일반 | 레어 | 금빛 | 전설 |
|------|------|------|------|------|
| 비시즌 평일 | 88% | 9% | 3% | 0% |
| 비시즌 주말 | 82% | 12% | 6% | 0% |
| 시즌 평일 | 78% | 10% | 8% | 4% |
| 시즌 주말 | 68% | 10% | 14% | 8% |

### 리롤 (다시 파기)

- 무료 리롤 2회 제공 (티어별 상이)
- API: POST `/mining/veins/reroll`
- 소진 시 안내 UI

### 아이디어 원석 생성

- 광맥 1개 선택 → 아이디어 10개 생성 (4군: stable×3, expansion×3, pivot×2, rare×2)
- API: POST `/mining/veins/{vein_id}/mine`
- 제목 + 요약 (한/영) + 키워드 콤보 + tier_type 표시
- 원하는 것을 금고에 반입 (PATCH `/ideas/vault`)

---

## 핵심 세션 루프

1. Mine 진입
2. 오늘의 광맥 3개 확인
3. 마음에 드는 조합 1개 선택
4. 아이디어 10개 생성
5. 원하는 아이디어 선택 → 금고 반입
6. (선택) 바로 개요 생성으로 이동
7. 다음 채굴 동기 확인

---

## UX 원칙

- 첫 화면은 무조건 "바로 채굴" 중심
- 상태 정보는 가볍게
- 카드 선택은 명확하게
- 일일 한도 소진 시 자연스러운 안내
