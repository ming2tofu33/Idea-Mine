# The Mine (홈 / 광산)

> 가장 중요한 첫 화면. "들어오면 바로 한 판 시작"이 이 공간의 역할이다.

---

## 공간 목표

- 3초 안에 오늘 할 일이 보인다
- 30초 안에 첫 결과를 얻을 수 있다
- 다시 들어오고 싶을 이유가 남는다

---

## 핵심 기능

### 오늘의 광맥 3개

- 매일 랜덤 키워드 조합 3세트 제공
- 가로 슬라이드(캐러셀) 형태로 넘기며 선택
- 각 카드에 키워드 5개, 희귀도, 분위기 태그 노출

**광맥 카드 구조:**
- 상단: 광맥 이름 (자동 생성 별칭)
- 중앙: 키워드 칩 5개
- 하단: 분위기 태그 + 희귀도

**광맥 이름 예시:** 감정 케어 광맥, 실버 헬스 광맥, 멀티모달 생산성 광맥

**분위기 태그 예시:** 실용형, 트렌디, 실험적, 시장성 높음, 감성 특화

**희귀도:** 일반 광맥(Common), 금빛 광맥(Golden), 전설 광맥(Legend)

**희귀도 결정 기준:**
- 일반 광맥: 기본 키워드 풀(109개)에서 조합. 상시 등장.
- 금빛 광맥: 분기 트렌드 키워드 1개 이상 포함. 분기 내내 등장.
- 전설 광맥: 시즌 한정 키워드 포함. 시즌 이벤트 기간(~1개월)에만 등장.

**등장 확률 (광맥 3개 각각 독립 적용):**

| 조건 | 일반 | 금빛 | 전설 |
|------|------|------|------|
| 비시즌 평일 | 97% | 3% | 0% |
| 비시즌 주말 | 94% | 6% | 0% |
| 시즌 평일 | 90% | 8% | 2% |
| 시즌 주말 | 82% | 13% | 5% |

**확률 고지:** 앱 내 광맥 화면 물음표(?) 아이콘 + 설정 > 확률 정보에 표시. 앱스토어 설명에도 명시. 한국 게임산업진흥법 + Apple/Google 정책 준수.

### 리롤 (다시 파기)

- 무료 리롤 2회 제공
- 짧은 셔플 애니메이션 후 새 광맥 등장
- 소진 시 보상형 광고로 추가 리롤

### 아이디어 원석 생성

- 광맥 1개 선택 -> 아이디어 draft 10개 생성
- draft는 가볍게: 제목 1줄, 설명 2~3줄, 핵심 타겟/가치 태그
- 그 중 원하는 것을 금고에 반입

---

## 홈 화면 구성 요소

1. 상단 상태바 (프로필, 연속 접속, 보유 재화, 알림)
2. 오늘의 광맥 3개 (핵심 CTA)
3. 리롤 버튼
4. 데일리 퀘스트 카드
5. 주말 특별 광맥 배너
6. 금고 미리보기
7. 하단 예고 구역 (공사 중 콘텐츠)

---

## 핵심 세션 루프

1. 홈 진입
2. 오늘의 랜덤 키워드 조합 3개 확인
3. 마음에 드는 조합 1개 선택
4. 선택한 조합 기반으로 아이디어 draft 10개 생성
5. 그 중 1개 선택
6. 선택한 1개에 대해 프로젝트 개요 문서 생성
7. 저장 또는 종료
8. 보상/퀘스트/다음 채굴 동기 확인

---

## UX 원칙

- 첫 화면은 무조건 "바로 채굴" 중심
- 상태 정보는 가볍게
- 카드 선택은 명확하게
- 보상과 저장은 보조 루프로 배치
- 공사 중 기능은 기대감만 남기고 복잡성은 줄이기
