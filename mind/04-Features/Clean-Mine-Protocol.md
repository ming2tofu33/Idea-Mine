# Clean Mine Protocol

> 광산을 안전하게 유지하고, 브랜드 톤을 지키며, 반복 악용을 방지하는 운영 시스템.

---

## 3가지 핵심 역할

1. **광산 안전 유지** -- 음란, 비속어, 혐오, 공격성이 서비스 경험을 오염시키지 않게 막기
2. **브랜드 톤 유지** -- 딱딱한 에러창이 아니라 세계관 안에서 자연스럽게 제지
3. **반복 악용 방지** -- 한 번 막는 것에서 끝나지 않고, 반복 시도는 실제 제재로 연결

> 가볍게 말하되 운영은 단단하게.

---

## MVP 차단 대상 (4가지)

| 유형 | 설명 |
|------|------|
| A. 음란/성적 표현 | 노골적 성행위, 성적 대상화, 성적 서비스 지향 조합 |
| B. 비속어/모욕 | 욕설, 인신공격, 노골적 비하 |
| C. 혐오/차별 | 특정 집단 공격/배제 표현 |
| D. 반복 스팸 | 의미 없는 반복, 도배, 같은 키워드 악용 |

---

## 내부 오염 등급

> 유저에게 노출 안 함. 내부 로직 전용.

| 등급 | 의미 | 대응 |
|------|------|------|
| Green | 문제 없음 | 통과 |
| Yellow | 표현 거칠음 | 수정 유도 (1단계) |
| Orange | 명확한 부적절 | 채굴 차단 (2단계) |
| Red | 반복 악용 | 쿨다운/제한 (3단계) |

---

## 4단계 대응 구조

### 1단계. 가벼운 오염 -- 수정 유도

**상황:** 약한 비속어, 애매한 표현, 의도 불명확

- 바로 막지 않고 수정 요청
- 대체 키워드 제안
- 채굴은 아직 시작 안 함

### 2단계. 명확한 오염 -- 채굴 차단

**상황:** 명백한 음란, 강한 욕설, 명확한 혐오

- 해당 채굴 즉시 차단
- 생성 결과 없음
- 금고/실험실로 진행 불가
- 관련 키워드 입력창에서 제거/마스킹

### 3단계. 반복 오염 -- 쿨다운

**상황:** 짧은 시간 안에 여러 번 차단, 유사 표현 반복, 우회 시도

- 일정 시간 채굴 제한
- 리롤, 생성, 개요 만들기 잠금

**제재 수위:**

| 차수 | 조치 |
|------|------|
| 1차 | 경고 |
| 2차 | 10분 쿨다운 |
| 3차 | 하루 일부 기능 제한 |
| 4차 | 공개 기능 제한 또는 관리자 검토 |

### 4단계. 공개 악용 -- 쇼케이스/거래소 제재

**상황:** 부적절한 프로젝트 공개, 혐오/음란 전시, 투자자 공간 악용

- 쇼케이스 등록 차단
- 게시물 숨김
- 신고 누적 시 공개 권한 박탈
- 심각한 경우 계정 검토

---

## 쇼케이스 등록 조건

- 프로젝트 개요 생성 완료
- 감정 완료
- 정책 위반 없음
- 반복 스팸 계정 아님

---

## 신고 사유 (Phase 4)

- 음란성
- 혐오/차별
- 욕설/모욕
- 스팸/도배
- 무관한 광고성 게시물

---

## "오염 감지" 연출

**추천:**
- 광석이 검게 변함
- 붉은 경고등 점등
- 작은 연기/노이즈 효과
- "오염 감지" 스탬프

**비추천:**
- 너무 큰 코믹 폭발
- 우스꽝스러운 실패 애니메이션
- 심각한 입력에도 장난처럼 보이는 연출

> 브랜드는 유지하되 정책은 가볍지 않게.

---

## MVP에서 꼭 넣을 것

- 음란/비속어/혐오 입력 차단
- 수정 유도 메시지
- 명확한 차단 메시지
- 반복 시도 쿨다운
- 쇼케이스 공개 전 최소 기준

## 2차 확장에서 넣을 것

- 신고 시스템
- 공개 콘텐츠 자동 숨김
- 신뢰 배지 (Badge-System 연동)
- 관리자 검토 큐
- 반복 우회 탐지 강화
