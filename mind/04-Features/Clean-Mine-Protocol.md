# Clean Mine Protocol

> 악성 입력과 공개 악용을 관리하는 신뢰 시스템. Phase 1에서 기본 1~3단계.

---

## 목적

- 악성 입력(스팸, 부적절한 키워드) 방어
- 공개 콘텐츠 품질 유지
- 성실한 활동을 배지 기반 신뢰도로 보상

---

## Phase 1 적용 범위 (1~3단계)

- 기본 입력 필터링
- 부적절한 키워드 감지
- 반복 스팸 방어

---

## 장기 확장 (Phase 4)

- 신고 시스템
- 신뢰 광산주 / 전시 승인 기획자
- 공개 기준 검수
- 공개 공간용 배지 노출
