# Badge System (배지 시스템)

> 활동의 "성격"을 보여주는 업적 체계 + 내부 신뢰 점수. 밖으로는 따뜻하고 명확하게, 안쪽은 엄격하게.

---

## 3층 신뢰 구조

| 층 | 유저 노출 | 설명 |
|----|----------|------|
| 1층. 활동 배지 | 프로필/쇼케이스에 표시 | 개별 업적 |
| 2층. 신뢰 등급 | 상태 문구로 표시 | 전체 활동 상태 |
| 3층. 내부 점수 | 노출 안 함 | 운영 판단용 |

> 숫자 점수는 공격적으로 느껴질 수 있어서, 유저에겐 배지와 등급만 보여줌.

---

## 배지 6종

### 꾸준함 계열

#### 1. 꾸준한 광부

- **의미:** 일정 기간 안정적으로 채굴한 유저
- **조건:** 최근 14일 중 7일 이상 활동, 또는 누적 채굴 세션 일정 수 이상
- **감정:** 꾸준함의 보상
- **시기:** MVP

#### 2. 연속 채굴자

- **의미:** streak 기반 연속 활동
- **조건:** 연속 채굴 3일 / 7일 / 14일 단계형 (단계마다 아이콘 변화 가능)
- **감정:** 재방문의 보상
- **시기:** 확장

### 문서화 계열

#### 3. 정제 완료 기획자

- **의미:** 원석을 개요 문서까지 자주 발전시키는 유저
- **조건:** 프로젝트 개요 일정 수 이상 생성 + 원석->개요->저장까지 이어진 문서 비율 반영
- **감정:** 결과물의 보상
- **시기:** MVP
- **중요:** 단순 생성 횟수만 세면 안 됨. 후속 행동까지 연결된 활동 반영

#### 4. 실험실 숙련자

- **의미:** 감정, 방향 정리, 문서 심화까지 꾸준히 한 유저
- **조건:** 감정 완료 횟수 기준 + 문서 심화 진행 횟수 + 일정 기간 반복 사용
- **감정:** 깊이의 보상
- **시기:** MVP

### 공개 신뢰 계열

#### 5. 신뢰 광산주

- **의미:** 신고 없이 안정적이고 품질 있게 공개 활동하는 유저
- **조건:** 일정 기간 신고 없음 + Clean Mine Protocol 반복 위반 없음 + 공개/저장 활동 안정적 + 쇼케이스 기준 통과 이력
- **감정:** 신뢰의 보상
- **시기:** 확장 (Phase 4 준비)

#### 6. 전시 승인 기획자

- **의미:** 쇼케이스 기준을 충족하는 프로젝트를 반복적으로 만든 유저
- **조건:** 쇼케이스 등록 기준 충족 프로젝트 n개 이상 + 공개 후 신고/숨김 이력 없이 유지 + 감정/정제까지 거친 문서만 인정
- **감정:** 공개 품질의 보상
- **시기:** 확장 (Phase 4)

---

## 악용 방지 보정

1. **단순 생성 도배는 인정 비중 낮게** -- 원석만 대량 생성해도 배지 빨리 못 따게
2. **후속 행동 비중 높게** -- 문서 완성/저장/감정 같은 의미 있는 활동에 가중치
3. **반복 위반 시 배지 획득 유예** -- Clean Mine Protocol 반복 위반이 있으면 일정 기간 신뢰 배지 획득 제한
4. **공개 배지는 비공개 활동과 분리** -- 공개 공간에서의 품질과 매너를 따로 평가

---

## 신뢰 등급

배지와 별개로 전체 상태를 보여주는 문구.

| 등급 | 의미 |
|------|------|
| 안정적 활동 | 기본 상태 |
| 신뢰 가능한 기획자 | 중간 |
| 검증된 광산주 | 최상위 |

- 프로필: `신뢰 상태: 안정적 활동`
- 쇼케이스: `검증된 광산주`

---

## 내부 신뢰 점수 (유저 비노출)

### 플러스 요소
- 정상 채굴
- 원석 저장
- 프로젝트 개요 생성
- 감정/정제 완료
- 신고 없는 공개 활동
- 쇼케이스 기준 통과

### 마이너스 요소
- Clean Mine Protocol 경고 누적
- 반복 차단
- 공개 후 신고 누적
- 스팸성 유사 프로젝트 반복
- 전시 숨김 조치

> 활동량 + 품질 + 안전성, 이 세 축으로 계산.

---

## 노출 방식

### 프로필
- 대표 배지 1개 크게 + 보조 배지 2개
- 예: `도민 광부님 | 정제 완료 기획자 · 꾸준한 광부`

### 쇼케이스 카드
- 작성자 아래 배지 최대 2개
- 신뢰 계열 배지 우선

### 거래소/투자자 화면
- 신뢰 광산주 / 전시 승인 기획자 우선 노출
- `최근 30일 안정 활동` 같은 추가 신호
- 배지가 단순 꾸미기가 아니라 **신뢰 신호**로 읽혀야 함

### 금고/실험실
- 프로필 상단 작은 배지
- "신뢰 이력이 쌓이고 있어요" 같은 짧은 안내

---

## 배지 획득 문구

- 꾸준한 광부 배지를 획득했어요.
- 정제 완료 기획자 배지가 열렸어요.
- 신뢰 광산주 조건에 한 걸음 더 가까워졌어요.
- 실험실까지 안정적으로 발전시킨 기록이 남았어요.

## 정상 활동 보상 문구

- 깨끗한 채굴 기록이 쌓이고 있어요.
- 광산 규칙을 잘 지키고 있어요.
- 신뢰 가능한 채굴 이력이 쌓였어요.

---

## MVP 기준

**포함:** 꾸준한 광부, 정제 완료 기획자, 실험실 숙련자 + 내부 신뢰 점수 + 배지 획득 조건 로직

**확장:** 신뢰 광산주, 전시 승인 기획자, 연속 채굴자 + 등급 시스템 + 거래소용 신뢰 신호 강화
