# Miner Level (광부 레벨)

> 채굴 활동에 따라 성장하는 50레벨 시스템. 레벨업마다 금고, 가방, 개요서, 핑크 결정, 배지 보상.

---

## 설계 원칙

- **자주, 조금씩** -- +2칸 같은 작은 보상을 자주 줘서 진행감 유지
- **보상 종류 로테이션** -- 매번 같은 보상이 아니라 4종이 돌아가며 등장
- **레벨 = 무과금 성장** -- Free 유저도 레벨로 금고/개요서가 늘어남. 티어 업그레이드와 별개의 성장축
- **마일스톤은 특별하게** -- 10레벨 단위에 배지 + 핑크 결정 동시 지급

---

## 보상 4종

| 종류 | 내용 | 빈도 |
|------|------|------|
| 금고 +2 | 금고 저장 용량 2칸 확장 | 매우 자주 (50레벨 중 ~30회) |
| 개요서 +1 | 일일 프로젝트 개요서 생성 한도 +1 | 5레벨 간격 (~8회) |
| 가방 +1 | 채굴당 반입 슬롯 +1 (Free만 해당, Lite/Pro는 광차 10 고정) | 7~8레벨 간격 (~5회) |
| 핑크 결정 | 프리미엄 재화 보너스 | 10레벨 마일스톤 (5회) |
| 배지 | 정체성 보상 (칭호) | 10레벨 마일스톤 (5회, 핑크 결정과 동시) |

---

## 5레벨 단위 패턴

```
Lv.X1: 금고 +2
Lv.X2: 금고 +2
Lv.X3: 가방 +1 또는 개요서 +1 (교대)
Lv.X4: 금고 +2
Lv.X5: 핑크 결정 (+ 10단위면 배지도)
```

## 티어별 누적 (Lv.50 기준)

| 항목 | Free | Lite | Pro |
|------|------|------|-----|
| 금고 기본 | 20 | 150 | 500 |
| 금고 Lv.50 | 20 + 58 = **78** | 150 + 58 = **208** | 500 + 58 = **558** |
| 가방 | 2 → **7** | 광차 10 (고정) | 광차 10 (고정) |
| 개요서/일 기본 | 1 | 5 | 20 |
| 개요서/일 Lv.50 | 1 + 5 = **6** | 5 + 5 = **10** | 20 + 5 = **25** |

금고와 개요서의 **레벨 보너스는 전 티어 동일** (+58, +5). 티어 간 차이는 기본값에서 나옴.

## 레벨업 조건

### XP 획득 행동

| 행동 | XP | 비고 |
|------|-----|------|
| 채굴 완료 | +10 | 일일 메인 행동 |
| 금고 반입 (아이디어 1개당) | +3 | 저장 유도 |
| 프로젝트 개요서 생성 | +15 | 파이프라인 진입 유도 |
| 감정 생성 | +10 | 파이프라인 깊이 유도 |
| 연속 접속 보너스 | +5/일 | 리텐션 보상 |

### 필요 XP 곡선

레벨이 올라갈수록 필요 XP가 완만하게 증가. 초반은 빠르게 올라가고 후반은 느려짐.

```
필요 XP = 기본값 + (레벨 × 증가폭)

Lv.1→2:  30 XP  (채굴 3번 분량)
Lv.10→11: 50 XP
Lv.25→26: 80 XP
Lv.40→41: 110 XP
Lv.49→50: 130 XP
```

총 누적 XP (Lv.50): 약 3,900 XP

Free 유저가 매일 채굴 1회 + 반입 3개 + 개요서 1회 = 약 34 XP/일.
Lv.50 도달까지 약 **115일 (4개월)** -- 장기 리텐션 목표와 일치.

```

```

```
"레벨 UP! Lv.19 → Lv.20"
"배지 획득! 🏅 베테랑 광부"
```
