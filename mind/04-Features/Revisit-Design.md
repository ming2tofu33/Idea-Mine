# Revisit Design (재방문 설계)

> 사용자가 다시 들어오게 만드는 장치들의 설계.

---

## 핵심 재방문 장치

| 장치 | 역할 |
|------|------|
| 데일리 랜덤 조합 | "오늘은 뭐가 나올까?" |
| 연속 접속 보상 | "하루라도 끊기 싫다" |
| 취향 광맥 해금 | "이 앱이 점점 나를 안다" |
| 주말 특별 광맥 | 이벤트성, 주말만의 기대감 |
| 금고 저장 | 남는 자산감 |
| 데일리 퀘스트 | 작은 목표 달성감 |
| 희귀 조합 발견 | 발견의 재미 |

---

## 주말 특별 광맥

- 평일: 티저 ("주말 특별 광맥이 준비되고 있어요")
- 주말: 활성 배너 ("이번 주말 한정: 미래 직업 광맥")
- 주말마다 테마가 달라야 반복 접속 이유가 생긴다

---

## 게임화에서 반드시 살릴 요소

1. **짧은 세션:** 30초~5분 안에 한 번의 보상
2. **즉시 보상:** 조합 -> 결과 바로 등장, 애니메이션/사운드
3. **가벼운 선택:** 슬롯머신처럼 고르기 쉽게
4. **수집 욕구:** 금고에 쌓이는 히스토리
5. **성장 감각:** 더 잘 캐고 있다는 느낌
